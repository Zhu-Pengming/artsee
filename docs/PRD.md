## 1. 产品概述

### 1.1 产品定位
**艺见心 (ArtSee)** 是全球领先的青年艺术人才成长全周期综合服务平台，连接艺术学习、创作、社交与商业变现。

### 1.2 目标用户
- **核心用户**：艺术留学意向学生（本科/研究生）
- **次级用户**：青年艺术家、艺术爱好者
- **商业用户**：艺术院校、画廊、收藏家、酒店/商业空间

### 1.3 产品愿景
成为全球领先的，服务于青年艺术人才成长全周期的综合性平台。

## 5. 竞品分析总结

### 5.1 主要参考竞品

| 产品 | 类型 | 借鉴点 | 适用模块 |
|------|------|--------|---------|
| **小红书** | 内容社区 | 瀑布流设计、标签体系、社区氛围 | 案例灵感、社区互动 |
| **Instagram** | 视觉社交 | 沉浸浏览体验、视觉优先 | 作品集展示 |
| **LinkedIn/Maimai** | 职业社交 | 模块化信息架构、认证体系 | 院校展示、导师认证 |
| **Grammarly** | AI 工具 | AI 辅助逻辑、渐进式引导 | AI 作品集助手 |
| **知乎** | 问答社区 | 专业问答生态、认证体系 | 互动服务 |
| **豆瓣小组** | 兴趣社区 | 圈层讨论、归属感建设 | 兴趣小组 |
| **Artsy** | 艺术电商 | 高清展示、专业信息 | 艺术交易 |
| **Artnet** | 艺术数据 | 价格指数、成交记录 | 拍卖数据服务 |
| **KLOOK/Viator** | 旅游预订 | 标准化流程、产品展示 | 文旅预订 |
| **一条/东家** | 生活方式 | 内容驱动、文化包装 | 文旅内容 |

### 5.2 差异化优势

1. **垂直领域深耕**：专注艺术留学细分赛道，而非泛泛的艺术社区
2. **全周期服务**：从学习到申请到职业发展的完整链路
3. **AI 深度集成**：不只是工具，更是创作伙伴
4. **商业闭环**：连接学习与变现，打通艺术人才成长路径

## 6. 技术架构要点

### 6.1 推荐技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端 (APP) | Flutter | 跨平台开发，UI 一致性 |
| 前端 (Web) | Next.js 15 | SSR/SSG，SEO 友好 |
| 后端 | Supabase | 数据库 + Auth + 存储 |
| AI 能力 | 第三方 API / 自研 | GPT-4、Stable Diffusion 等 |
| 推送 | Firebase / 极光 | 消息推送 |
| 支付 | Stripe / 支付宝 / 微信 | 多币种支持 |

### 6.2 核心数据模型

- **Users** - 用户基本信息与角色
- **UserProfiles** - 扩展资料（含 role: user/admin/moderator）
- **Schools** - 院校信息
- **Programs** - 课程/项目信息
- **Portfolios** - 作品集
- **Posts** - 社区内容
- **Questions** - 问答
- **Artworks** - 交易作品
- **Events** - 文旅活动
- **Applications** - 申请记录

## 8. 附录

### 8.1 设计原则

1. **视觉优先**：艺术内容需要高质量的视觉呈现
2. **简约克制**：让内容本身成为焦点，UI 不喧宾夺主
3. **专业可信**：建立权威感，获得用户信任
4. **温暖人文**：艺术是感性的，产品要有温度

### 8.2 术语表

| 术语 | 说明 |
|------|------|
| Feed | 信息流，瀑布流式内容展示 |
