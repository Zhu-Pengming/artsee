# 维也纳应用艺术大学介绍

## TL;DR
维也纳应用艺术大学是一所位于奥地利维也纳的公立艺术类高等院校，拥有悠久的历史和丰富的艺术教育资源。学校提供从学士到博士的多个学位课程，涵盖建筑、造型艺术、工业设计、媒体设计等多个领域。学校在国际上享有盛誉，与多所国际机构建立了合作关系，并多次获得各类荣誉。

## 关键声明
- **历史与背景**：维也纳应用艺术大学成立于1867年，其历史可追溯至1493年创立的维也纳美术艺术学院。学校在19世纪参与了维也纳城市规划建筑革新，并在二战期间关闭，1988年重建。
- **学生与师资**：学校现有约2000名学生，其中四分之一来自外国。有200多名教授和老师在校任教。
- **专业设置**：学校设有建筑、造型艺术、工业设计、媒体设计等专业，提供学士、硕士、博士三级学位课程。
- **学术研究**：学校的研究输出包括书籍章节，知名学者如Mario Carpo担任建筑研究所的建筑理论教授。
- **国际合作**：维也纳应用艺术大学与多所国际机构建立了合作关系，涵盖学生交换、教职人员交流、合作研究及共建联合实验室等领域。
- **入学条件**：入学考试每年一次，外国留学生还需具备一定德语水平。
- **荣誉与认证**：学校获得中国教育部留学服务中心认证，是奥地利的热门留学院校，在2023年QS艺术与设计专业世界排名中位列第46名。

## 详细分析

### 学生与师资
学校现有约2000名学生，其中四分之一来自外国。有200多名教授和老师在校任教。这一数据表明，学校拥有庞大的学生群体和优秀的师资力量，为学生提供了良好的学习环境。

### 学术研究
