# 圣卡洛斯大学艺术学院简介

## TL;DR
圣卡洛斯大学（University of San Carlos，简称USC）是菲律宾和亚洲最古老的私立大学之一，成立于1565年。学校位于菲律宾第二大城市宿务市，提供近40个本科专业和83个硕士和博士专业。本文将详细介绍圣卡洛斯大学的历史、院系设置、专业介绍、荣誉奖项以及与中国高校的合作情况。

## 关键声明
- 圣卡洛斯大学成立于1565年，是菲律宾和亚洲最古老的私立大学之一。
- 学校位于宿务市，提供近40个本科专业和83个硕士和博士专业。
- 学校拥有近2万名学生，拥有8个学院，4个校区。
- 圣卡洛斯大学是菲律宾前10强大学，最早与中国高等院校合作的菲律宾高校之一，并获得中国教育部认证。
- 学校拥有多个学院，包括商学院、教育学院、法学院等。
- 学校获得菲律宾高等教育委员会（CHED）自治地位的认可。
- 学校的10个项目获得了FAAP的II级认证。
- 学校提供丰富的本科和研究生课程，涵盖多个领域。
- 学校在2017年普莱斯传媒法模拟法庭亚太赛区选拔赛中获得冠军。
