# Queen's University (女王大学/皇后大学) Wiki —— 决策型版本

## TL;DR
- 女王大学是一所位于加拿大安大略省Kingston的综合研究型大学，以其商学院、医学院、工程学院和法学院而闻名。
- 学校的艺术与人文领域，特别是艺术史与艺术保护系，是加拿大顶尖的。
- 然而，传统的BFA Visual Art项目自2023年2月起暂停招生，新项目仍在审批中。
- 学校提供多种艺术相关项目，包括艺术史、艺术保护、电影与媒体、戏剧与音乐等。
- 对于艺术史、艺术保护、跨学科创意或音乐戏剧路线的申请者，女王大学是一个合理的选择。

## Key Claims

### 1. 学校概况
- **地理位置**：加拿大安大略省Kingston
- **学校类型**：综合性研究型大学（U15成员，非独立艺术学院）
- **重点科系**：艺术史与艺术保护系、电影与媒体系、戏剧与音乐系
- **学校特色**：艺术史与艺术保护系是加拿大顶尖，提供加拿大唯一的艺术保护硕士项目。

### 2. BFA Visual Art项目暂停
- **项目状态**：暂停招生（自2023年2月起）
- **原因**：质量保证和可持续性问题
- **影响**：2026年秋及以后入学：不能申请BFA Visual Art；想读Queen's Visual Arts的本科生：等新的BAH Visual Arts启动（时间未定，大概率2027年或更晚）

### 3. 艺术相关项目
- **艺术史**：提供本科、硕士和博士项目，涵盖全球艺术传统和专题研究。
- **艺术保护**：提供加拿大唯一的艺术保护硕士项目，跨学科领域。
- **电影与媒体**：提供电影与媒体学士学位，强调媒体制作和动画。
- **戏剧与音乐**：提供戏剧、音乐和音乐戏剧学士学位。
- **跨学科创意**：提供媒体与表演制作学士学位，涵盖AR/VR、沉浸式叙事、数字表演。

### 4. 申请信息
- **英语水平要求**：IELTS 6.5 / TOEFL iBT 88+（学校层级要求；部分专业更高）
- **申请截止日期**： OUAC申请：2月1日（艺术与科学）；MA Art History：1月10日
- **学费**：国际生本科学费约CAD 57,000/年；研究生学费约CAD 15,000/年

### 5. 校园设施与创作资源
- **艺术设施**：Agnes Etherington艺术中心、Isabel Bader表演艺术中心、艺术图书馆、BFA工作室
- **跨学科资源**：电影与媒体系、戏剧与音乐系、Queen's School of Computing

### 6. 学生体验
- **小班教学**：BFA项目历史上每届约30人，四年同一群人在一起学习。
- **艺术社区**：与RISD、SAIC等纯艺术学院相比，Queen's的艺术社区较小，但更多元化。
- **城市环境**：Kingston是一个历史小城，距离Toronto/Ottawa各约2.5小时车程，适合喜欢安静、专注创作的人。
