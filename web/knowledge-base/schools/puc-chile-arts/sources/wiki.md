# 智利天主教大学艺术学院

## TL;DR
智利天主教大学艺术学院是智利天主教大学的一部分，成立于1888年，是一所私立宗座大学。该学院提供广泛的学科和专业，包括艺术和文学。智利天主教大学在拉丁美洲享有盛誉，是智利顶尖的高等教育机构之一。本文将分析该学院的成立背景、学科设置、学术排名、著名校友以及与其他大学的合作关系。

## 备注/分析
### 成立背景
智利天主教大学成立于1888年，由天主教会创办，是一所私立宗座大学。该大学的成立旨在培养具有社会责任感和专业素养的人才，以促进智利社会的进步和发展。

### 学科设置
智利天主教大学艺术学院提供广泛的学科和专业，包括艺术和文学。这些学科涵盖了从本科到研究生的各个层次，为学生提供了丰富的学习选择。

### 学术排名
根据QS世界大学排名拉丁美洲地区排名2011/2012，智利天主教大学是南美第2、智利第1的大学。这表明该大学在拉丁美洲享有盛誉，是智利顶尖的高等教育机构之一。

### 著名校友
智利天主教大学培养了许多杰出校友，其中最著名的是第35任智利总统塞巴斯蒂安·皮涅拉。这些校友在各自的领域取得了卓越的成就，为智利社会做出了重要贡献。

### 国际合作
智利天主教大学与多所国际知名大学有合作关系，包括巴黎第六大学、北京交通大学等。这些合作关系为学生提供了更多学习和交流的机会，有助于提升学院的学术水平和国际影响力。
