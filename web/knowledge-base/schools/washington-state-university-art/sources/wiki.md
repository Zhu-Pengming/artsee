# 华盛顿州立大学 - 维基百科

## TL;DR
华盛顿州立大学（Washington State University, WSU）是一所位于美国华盛顿州的研究型大学，成立于1890年，是美国赠地大学之一。该校拥有多个校区，主校区位于普尔曼，提供多样化的本科和研究生课程。WSU在全球和全国范围内都有较高的排名，培养了众多杰出校友。

## 详细分析

### 学校概况
华盛顿州立大学（Washington State University, WSU）是一所位于美国华盛顿州的研究型大学，成立于1890年，是美国赠地大学之一。该校拥有多个校区，主校区位于普尔曼，而在斯波坎、温哥华及三城也设有分校。

### 学校历史
WSU的前身是华盛顿州农业学院、实验站和科学学院，成立于1890年3月28日。作为赠地大学，美国联邦政府及华盛顿州州政府于美国西北部拨出769平方公里（19万公顷）的土地给大学作为农业及科学的研究。1893年，大学开始提供文科的课程，此后逐渐发展成为综合性的研究型大学。

### 学校规模
WSU拥有约28,581名学生，其中本科生24,728名，研究生4,269名。学校拥有2,261名教师，其中包括947名博士生。

### 学校排名
WSU在全球和全国范围内都有较高的排名。根据2021年的数据，WSU在全球排名301-400名，在美国排名176名。这反映了其学术水平和教学质量。

### 学校特色
WSU拥有多个校区，分布在华盛顿州的不同城市，为学生提供了多样化的学习环境。学校的主校区位于普尔曼，依山而建，风景优美。校园内的主要建筑物都是红色的砖墙，这是因为在校园内的第一座建筑物为红砖的建筑，自此以后所有的建筑的仿效类似的风格。

### 学校运动
WSU是太平洋十大学联会的成员之一，体育运动的吉祥物是美洲狮，故华盛顿州立大学的学生一般被称为“华盛顿州美洲狮”。学校参与的运动包括美式橄榄球、棒球、篮球、高尔夫球、田径和女子排球等，其中以美式橄榄球最受注目。

### 学校校友
WSU培养了众多杰出校友，在各个领域取得了显著成就。其中，保罗·艾伦是微软的创立者之一，盖瑞·拉尔森是著名的漫画家，其他知名校友还包括爱德华·默罗、舍曼·亚历克西、菲力普·艾贝尔森、派蒂·莫瑞、蒂莫西·利里、道夫·龙格尔、王作荣、克雷·汤普森和欧文·罗斯等。

## 结论
华盛顿州立大学是一所历史悠久、规模庞大、学术水平较高的研究型大学。该校拥有多个校区，提供多样化的本科和研究生课程，培养了众多杰出校友。在全球和全国范围内，WSU都有较高的排名，是学生追求学术和职业发展的理想选择。
