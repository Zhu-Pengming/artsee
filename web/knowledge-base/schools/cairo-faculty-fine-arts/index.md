# Faculty of Fine Arts, Helwan University (Cairo)

## 学校概况 / Overview

**Faculty of Fine Arts, Helwan University**（赫勒万大学美术学院），中文通称"埃及开罗美术学院"，是**1908年创办**的中东第一所美术学校，位于开罗核心文化区Zamalek。提供建筑、绘画、雕塑、装饰、图形艺术、艺术史六大方向的本科和研究生项目。

**核心定位**: 综合型艺术与建筑学院，法语"École des Beaux-Arts"体系在埃及的体现。

## 专业设置 / Programs

### 六大系所

1. **建筑系 (Architecture)**: 强调建筑文化遗产与现代可持续设计
2. **绘画系 (Painting)**: 培养当代绘画艺术家，全埃及享有盛誉
3. **雕塑系 (Sculpture)**: 该地区历史最悠久的雕塑教育基地
4. **装饰/室内设计系 (Decoration)**: 专注空间创新室内设计
5. **平面设计系 (Graphic)**: 传统书籍装帧、版画、印刷品设计及数字媒体
6. **艺术史系 (Art History)**: 仅研究生层级

### 学位阶段
- 本科: Bachelor
- 研究生: Diploma, Master's, Doctoral

## 申请信息 / Admissions ⚠️

**⚠️ 核心挑战**: 公开英文官网中未找到学院级的完整申请清单

**已知要求** (来源第三方，需验证):
1. **能力测试**: 埃及高等教育部统一组织
2. **作品集**: 10-20件原创作品（素描、油画、雕塑等）
3. **语言**: 主要为阿拉伯语，部分研究生项目可能要求英语（雅思6.0/托福80+）
4. **申请时间**: 通常1月开放，部分项目3月15日截止

**申请流程**: Study in Egypt平台 + 大学国际学生办公室

## 学费与资助 ⚠️

**学费（估算，非官方）**:
- 国际生本科: $3,500-4,500/年
- 国际生研究生: $4,500/年
- 一次性杂费: ~$3,500

**奖学金** (2025/26):
- 60个名额
- 不覆盖生活费、住宿、机票、保险、签证、教材

## 设施资源

- 建筑、绘画、雕塑、装饰、图形艺术工作室
- Studios、workshops、galleries
- 学院博物馆（2016年开放）
- **FOFA Comics Exhibition**: 60名学生参与，主题覆盖社会问题

## Zamalek区位优势

- 开罗核心文化区
- 周边画廊、博物馆林立
- 多元文化碰撞
- 生活成本: $2,000-3,000/月（但实际可更低）

## 适合谁

### 适合
- 追求古典艺术传统和扎实基本功训练
- 希望沉浸式体验阿拉伯文化、伊斯兰艺术
- 追求高性价比
- 对公共艺术、大型雕塑、壁画有兴趣

### 不适合
- 追求前沿数字媒体艺术和当代批判理论
- 需要纯艺术学院"studio氛围"
- 不计划学习阿拉伯语，希望全程英语授课
- 看重QS综合排名光环
