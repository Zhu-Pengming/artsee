# 北卡罗来纳大学教堂山分校简介

## TL;DR
北卡罗来纳大学教堂山分校（UNC）是一所位于美国北卡罗来纳州的公立研究型大学，成立于1789年，是美国历史上第一个在十八世纪对外招生的公立大学。UNC在2025年U.S.News美国最佳大学排名中位列第27位，世界大学排名第26位。学校拥有多个学院，提供广泛的本科和研究生课程，包括医学、法学、商学、工程学、文学等。UNC以其卓越的学术研究、丰富的校园文化和众多知名校友而闻名。

- **学院与专业**：UNC拥有多个学院，提供广泛的本科和研究生课程，包括医学、法学、商学、工程学、文学等。
- **学术研究**：UNC在多个领域有着卓越的学术研究，包括医学、法学、商学、工程学、文学等。

- **知名校友**：UNC培养了众多知名校友，包括诺贝尔奖得主、普利策奖得主、罗德学者等。

## 详细分析
### 历史与排名

## 其他信息
- **地理位置**：UNC位于美国北卡罗来纳州，交通便利，距离达勒姆机场20英里，Durham火车站8英里。
- **校园环境**：UNC拥有优美的校园环境，包括老井、Morehead-Patterson钟塔、McCorkle广场等标志性建筑。
- **住宿**：UNC提供32座宿舍楼，分为13个居住区，每个居住区都有管理人员和RA常驻，帮助学生解决学习和生活上的困难。
- **餐饮**：UNC的Meal Plans主要包括两种：Unlimited和按学期计算的餐次。
- **基础设施**：UNC拥有设施齐全的现代教学大楼、校舍配套设施、研究室、体育馆、医院等。
- **知名校友**：UNC培养了众多知名校友，包括美国总统、美国副总统、美国州长、美国国会议员、诺贝尔奖得主、普利策奖得主、罗德学者等。
