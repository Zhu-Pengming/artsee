# 康考迪亚大学介绍

## TL;DR
康考迪亚大学是位于加拿大蒙特利尔的公立综合类大学，由两所百年老校合并而成。学校拥有约4.6万在校学生，来自全球130多个国家，是加拿大办学规模最大和国际化程度最高的高等院校之一。康大设有100多个本科专业及100多个硕博专业，授课语言为英文。其商学院、工程与计算机科学学院、文理学院、艺术学院等在加拿大享有盛誉。本文将详细介绍康考迪亚大学的办学历史、院系专业、学科建设、教学建设、学校排名、学校环境、知名校友等方面。

## 关键信息

### 办学历史
- 康考迪亚大学成立于1974年，由乔治威廉姆斯大学和圣徒学院合并而成。
- 学校拥有约4.6万在校学生，来自全球130多个国家。
- 是加拿大办学规模最大和国际化程度最高的高等院校之一。

### 院系专业
- 商学院：约翰墨森商学院（JMSB）是加拿大第一个经过世界精英商学院协会（AACSB）认证的商学院。
- 工程与计算机科学学院：吉娜科迪工程和计算机学院（ENCS）是加拿大最好的工程学院之一。
- 文理学院：地处环境优美的Loyola校区，师资力量雄厚，教学和实验设备先进。
- 艺术学院：覆盖艺术教育，艺术历史，片场艺术，视觉艺术等领域。

### 学科建设
- 商学院：金融专业排名全加连年第一，会计专业也是多次高居排名榜首。
- 工程与计算机科学学院：所有专业均获得了“加拿大工程认证机构（CEAB）”和“美国工程技术认证机构（ABET）”的认证。
- 文理学院：新闻学和传媒学作为其强势学科，在加拿大享有很好的声誉。
- 艺术学院：梅尔·霍普海姆电影学院是加拿大最大的高校电影研究中心。

### 教学建设
- 学校非常注重教学与社会需求的结合，致力于培养有丰富知识和技能并很快能够独当一面的人才。
- 与各行业的公司建立了紧密的关系，为学生提供广泛的就业市场。
- 注重国际上学院的联系，发展国际学院合作中心。

### 学校排名
- 加拿大综合排名：位列加拿大综合类大学前十。
- QS世界大学排名：2024年位列387位。
- 泰晤士高等教育世界大学排名：2016年位列418位。

### 学校环境
- 蒙特利尔是加拿大历史最悠久的城市，世界第二大法语城市，且被评为全球最安全的城市之一。
- 蒙特利尔是个国际大型金融中心，在2014年的全球金融中心排名中位列第16位。
- 蒙特利尔是个极具文化底蕴的城市，被《Monocle》杂志评为“加拿大文化之都”。
- 蒙特利尔是个饱含艺术气息的城市，被UNESCO评为“最佳设计城市”。
