# 康考迪亚大学维基百科页面

## TL;DR
康考迪亚大学是加拿大魁北克省蒙特利尔市的一所公立英语研究型大学，成立于1974年，由洛约拉学院和乔治·威廉姆斯爵士大学合并而成。该大学拥有两个校区，提供超过400个本科专业和120个研究生专业。本文分析了康考迪亚大学的历史、学术结构、校园设施、学生生活、排名和声誉等方面。

## 关键声明
- 康考迪亚大学是加拿大魁北克省蒙特利尔市的一所公立英语研究型大学。
