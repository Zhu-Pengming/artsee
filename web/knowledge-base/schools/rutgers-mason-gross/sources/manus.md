# 罗格斯大学梅森·格罗斯艺术学院 - 决策型 Wiki v1.0

## TL;DR
本文档是对罗格斯大学梅森·格罗斯艺术学院（Mason Gross School of the Arts, MGSA）的全面概述，基于2024-2026年的公开信息。MGSA是罗格斯大学下属的独立艺术学院，以其MFA Visual Arts项目而闻名，该项目的fully-funded奖学金吸引了众多申请者。本文档详细介绍了学校的概况、项目、申请要求、学费、奖学金、校园设施、周边艺术资源、校友故事、就业去向以及针对中国学生的特别信息。

## 关键声明

### 学校概况
- MGSA是罗格斯大学下属的独立艺术学院，提供多种艺术和设计项目。
- 学校位于新泽西州新布鲁斯威克，距离纽约市约45分钟车程。
- MGSA以其MFA Visual Arts项目而闻名，该项目提供fully-funded奖学金。

### 项目
- MGSA提供多种项目，包括BFA、BA、MFA Visual Arts、MFA Design、Filmmaking、Theater、Dance和Music。
- MFA Visual Arts项目是美国第一个non-disciplinary-specific的MFA项目，以其当代艺术史意义而著称。

### 申请要求
- 申请者需要提交大学申请、成绩单、推荐信、作品集或设计作品、个人陈述和申请费。
- 英语语言要求为TOEFL iBT 83分或IELTS 7.0分。

### 学费与奖学金
- MGSA的学费根据学生是否为州内居民而有所不同。
- MFA Visual Arts项目的所有录取者都将获得fully-funded奖学金，覆盖学费。
- MFA Design项目提供partial scholarship和part-time工作机会。

### 校园设施与创作资源
- MGSA提供多种校园设施，包括工作室、画廊、图书馆和跨学科资源。
- 学生可以访问Zimmerli Art Museum、Rutgers Makerspace和Rutgers Art Library等资源。

### 周边艺术资源
- MGSA位于新泽西州，靠近纽约市，学生可以轻松访问该市丰富的艺术资源。

### 校友故事与就业去向
- MGSA的校友包括Alice Aycock、Cheryl Dunye、Amy Feldman等知名艺术家。
- 毕业生通常进入自由职业艺术家、画廊director、高校lecturer/professor、博物馆curator等领域工作。

### 中国学生视角
