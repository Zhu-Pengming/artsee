# 罗格斯大学介绍

## TL;DR
罗格斯大学，全名新泽西州立罗格斯大学，是美国新泽西州规模最大的高等学府，也是一所公立研究型大学。学校成立于1766年，是美国独立战争前建立的九所殖民地学院之一。罗格斯大学是美国大学协会成员，拥有29个学院，300多个科研中心和研究所，提供150多个本科专业及400多门研究生课程。学校在数学、物理、统计学、信息科学、计算机科学等领域拥有一批世界著名的学者与研究员，具有强大的研究实力。

## 关键信息

### 学校概况
- **成立时间**：1766年
- **原名**：皇后学院
- **现名**：罗格斯大学
- **位置**：美国新泽西州
- **类型**：公立研究型大学
- **成员**：美国大学协会
- **特色**：公立常春藤，十大联盟成员

### 发展历程
- 1766年，皇后学院成立。
- 1825年，学院更名为罗格斯学院。
- 1864年，罗格斯学院成为州立赠地学校，正式成立罗格斯科学学院。
- 1924年，罗格斯科学学院更名为罗格斯大学。

### 学校规模
- **校区**：新不伦瑞克校区、纽瓦克校区、卡姆登校区
- **学生人数**：67620人（截至2024年6月）
- **教师人数**：10000余名
- **科研中心和研究所**：300多个

### 学科设置
- **本科专业**：150多个
- **研究生课程**：400多门
- **学院数量**：29个

### 研究实力
- 在数学、物理、统计学、信息科学、计算机科学等领域拥有一批世界著名的学者与研究员。
- 拥有多个科研中心和研究所，研究资金雄厚。

### 荣誉与排名
- 被评为新泽西州最佳雇主之一。
- 在多个国际大学排名中位列前茅。
