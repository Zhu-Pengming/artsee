# 墨西哥国立自治大学艺术与设计学院简介

## TL;DR
墨西哥国立自治大学（UNAM）艺术与设计学院是墨西哥和拉丁美洲地区历史最悠久、规模最大的综合性大学之一，提供广泛的学术课程和研究项目。该学院位于墨西哥城，拥有丰富的历史和文化遗产，以及现代化的校园设施。本文将详细介绍UNAM艺术与设计学院的历史沿革、院系设置、学术排名、优秀校友、建筑特色、基本资料、世界遗产描述和社会评价。

## 关键声明
- UNAM成立于1551年，是墨西哥和拉丁美洲地区历史最悠久、规模最大的综合性大学之一。
- 学院位于墨西哥城，提供从学士学位到博士学位的上百种课程。
- 校园内拥有中央图书馆、奥林匹克体育馆等标志性建筑，外墙壁画由迭戈·里维拉等艺术家创作。
- UNAM培养出三位诺贝尔奖得主，涵盖和平奖、文学奖及化学奖领域。
- 学院拥有丰富的历史和文化遗产，以及现代化的校园设施。
- 校园体现出独特的20世纪现代风格，融合了城市化、建筑、工程、景观设计和艺术。

## 备注/分析
### 历史沿革
- UNAM的前身为1551年西班牙王室批准设立的墨西哥皇家大学，历经多次更名后于1929年获得自治权并定现名。
- 1949年至1952年，大学城核心校区建成，由60多名建筑师、工程师及艺术家联合设计，2007年因融合现代建筑与传统元素被列入《世界遗产名录》。
- 学院拥有丰富的历史和文化遗产，是墨西哥乃至拉丁美洲地区的重要学术机构。

### 院系设置
- UNAM拥有14个系、交叉学科部、4个专业学院、文理学院、9个预科学校、函授部、自然科学研究协调委员会、人文科学研究协调理事会、文化传播协调委员会、167个图书馆以及出版社。
- 艺术与设计学院下设多个专业，包括建筑学、自然科学、政治-社会科学、会计与管理、法律、经济学、哲学与文学、工程学、兽医学与动物饲养学、牙科学、心理学和化学等。

### 学术排名
- 2024年QS世界大学排名位列全球第93位。
- 在考古学、建筑学、艺术与设计等领域，UNAM在QS世界大学专业排名中表现突出。

### 优秀校友
- UNAM培养出三位诺贝尔奖得主，涵盖和平奖、文学奖及化学奖领域。
- 学院还培养出许多著名教育家、科学家、艺术家和学者。

### 建筑特色
- UNAM校园体现出独特的20世纪现代风格，融合了城市化、建筑、工程、景观设计和艺术。
- 校园内拥有许多著名建筑，如韦华拉塔楼、中心图书馆和大学奥运体育馆。
