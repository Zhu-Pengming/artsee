# 比利时安特卫普皇家美术学院官方入学要求

## TL;DR
比利时安特卫普皇家美术学院（Ravca）为有意向在视觉艺术领域深造的学生提供了详细的入学要求。本文档分析了该学院的入学条件、语言要求、艺术入学考试流程、学费、财务援助和特殊需求学生支持等关键信息。

## 关键声明
- **入学条件**：Ravca为比利时、荷兰和其他国家/地区的学生提供视觉艺术学士和硕士学位课程。
- **语言要求**：学生必须具备荷兰语或英语的语言能力，达到B2水平（CEFR）。
- **艺术入学考试**：所有申请者必须通过艺术入学考试，包括预选和艺术作业。
- **学费**：学费因学生类型和居住地而异，欧盟学生和非欧盟学生费用不同。
- **财务援助**：提供多种奖学金和财务援助选项，包括国际奖学金和弗拉芒社区奖学金。
- **特殊需求学生支持**：Ravca为有特殊需求的学生提供便利措施。

## 入学条件
- **学位要求**：比利时学生需具备中等教育文凭或同等学历；荷兰学生需具备VWO文凭或HBO预科；其他国家/地区学生需具备本国大学入学资格。
- **中国学生**：中国学生需通过APS审核，并获得APS证书，才能申请Ravca。
- **学历认证**：非欧盟学生需提供学历认证。

## 语言要求
- **荷兰语**：学生需具备B2水平的荷兰语能力，可通过完成至少一年的荷兰语中等教育、获得至少60个荷兰语高等教育学分或通过B2水平语言测试来证明。
- **英语**：学生需具备B2水平的英语能力，可通过以下途径证明：ASO文凭、VWO文凭、英语中等教育或高等教育文凭、英语语言测试证书（如IELTS、TOEFL、Cambridge English等）。

## 艺术入学考试
- **预选**：申请者需提交数字作品集，预选委员会将根据作品集进行筛选。
- **艺术作业**：通过预选的申请者需参加艺术作业，包括绘画、雕塑、摄影等。
- **评估标准**：评估学生的艺术过程、创造力、想象力、文化素养、艺术/设计兴趣、视觉文化和绘画技能。

## 学费
- **欧盟学生**：根据奖学金类别（奖学金、接近奖学金、非奖学金）和学分合同类型（学位和/或学分合同、考试合同）收取不同学费。
- **非欧盟学生**：学费较高，包括学位和/或学分合同和考试合同。
- **课程相关费用**：包括书籍、材料、工作室费用等。

## 财务援助
- **国际奖学金**：提供多种国际奖学金，包括弗拉芒社区奖学金、欧盟奖学金等。
- **弗拉芒社区奖学金**：针对非欧盟学生，需满足特定条件。
- **AP大学学院财务援助**：仅适用于欧洲经济区学生，需满足特定条件。

## 特殊需求学生支持
- **便利措施**：为有特殊需求的学生提供便利措施，如学习辅助、考试调整等。
- **申请流程**：学生需尽早申请便利措施，并在截止日期前提交申请。

## 学术规则和日历
- **教育考试规则**：规定教育、考试和学生法律地位的相关规则。
- **学术日历**：包括课程、考试、补考、成绩公布、咨询和反馈、毕业典礼和假期等重要日期。
