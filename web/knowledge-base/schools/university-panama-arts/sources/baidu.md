# 巴拿马大学简介

## TL;DR
巴拿马大学是巴拿马共和国第一所官方认可的高等教育机构，成立于1935年，拥有多个校区。学校以培养国家专业人才为使命，在国际上有一定影响力。本文将分析巴拿马大学的历史、规模、教育合作以及其在国际事务中的立场。

## 关键声明
- 巴拿马大学是巴拿马共和国第一所官方认可的高等教育机构，成立于1935年。
- 学校拥有75,156名学生，其中国际学生2,462人。
- 巴拿马大学在QS世界大学排名中位列1401+位，拉丁美洲及加勒比地区排名第181-190位。
- 学校强调培养国家专业人才，毕业生中多人曾担任巴拿马总统等高层职位。
- 巴拿马大学与中国保持教育合作，设有孔子学院。
- 学校在2025年4月22日发表声明，谴责美国政府干涉巴拿马运河事务。
- 学校在2025年12月25日批评日本首相高市早苗的涉台错误言论和日本官员的‘拥核’言论。

### 国际排名与影响力
