# 爱丁堡玛格丽特女王大学

## TL;DR
爱丁堡玛格丽特女王大学（Queen Margaret University，简称QMU）是一所位于英国苏格兰爱丁堡的公立综合性大学，成立于1875年。该校以职业和专业教育为重点，学科涵盖商科、管理与信息、舞台艺术和保健等领域。QMU在2025年国家学生调查中在爱丁堡高等教育机构中整体学生满意度排名第一，拥有良好的学术声誉和丰富的校园设施。

## 关键声明
- QMU成立于1875年，是一所位于苏格兰爱丁堡的公立综合性大学。
- 学校以职业和专业教育为重点，学科涵盖商科、管理与信息、舞台艺术和保健等领域。
- 2008年，QMU在新加坡开设亚洲校区。
- 在2025年国家学生调查中，QMU在爱丁堡高等教育机构中整体学生满意度排名第一。
- 学校在多个学科领域拥有较高的排名，包括商业管理、放射学、旅游课程等。
- QMU提供多种学位和文凭课程，包括本科、硕士和研究生文凭。
- 学校设施包括计算机房、图书馆、游泳池、球场、银行、商店和宿舍楼。
- QMU为国际学生提供全面的学术、医疗、就业和咨询服务。
