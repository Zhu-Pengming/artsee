# 爱丁堡玛格丽特女王大学（Queen Margaret University, Edinburgh / QMU）深度调研报告

## TL;DR

- QMU 是一所小而专的苏格兰公立现代大学，前身可追溯至 1875 年的"爱丁堡烹饪与家政学校"，2007 年正式升格为大学。三大旗舰领域是健康与康复、创意与文化、可持续商业，王牌专业是言语与语言治疗和戏剧表演。
- QMU 综合排名属英国中下游，但戏剧/PR/医疗治疗类专业排名远高于综合排名。对于想做演员/影视/PR/服装设计/言语治疗等"动手就业型"专业，且预算有限的学生，QMU 是性价比极高且中国教育部正常认证的稳妥选择。
