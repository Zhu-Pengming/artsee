# 阿尔伯塔大学简介

## TL;DR
阿尔伯塔大学（University of Alberta，简称UA）是加拿大的一所公立综合类研究型大学，成立于1908年，位于阿尔伯塔省埃德蒙顿市。该校是加拿大U15研究型大学联盟创始成员，世界大学联盟成员，世界能源大学联盟成员。阿尔伯塔大学以其在地球科学、石油化工、化学、农学、生物医学等学科的研究而闻名，人工智能专业在全球居于领先地位。该校提供200多个本科和500多个研究生专业课程，拥有来自世界159个国家的46201名学生。
