# 加州州立大学北岭分校 - 维基百科

## TL;DR
加州州立大学北岭分校（California State University, Northridge，简称CSUN或Cal State Northridge）是一所位于美国加利福尼亚州洛杉矶的公立大学，成立于1958年。该校是加州州立大学系统内入学人数第三多的学院，以计算机工程、土木工程闻名。本文将详细分析该校的历史、学术成就、学生人数、校园设施等信息。

## 关键声明
- **成立时间**：1958年
- **学校类型**：公立大学
- **入学人数**：38,310（2013年秋）
- **本科生人数**：33,771（2013年秋）
- **研究生人数**：4,912（2013年秋）
- **博士生人数**：123（2013年秋）
- **校长**：Dianne F. Harrison
- **教师人数**：1,700
- **捐赠基金**：$8750万（2014）
- **校区面积**：353英亩（143公顷）
- **排名**：2021年《美国新闻与世界报道》将其排在“西部地区大学”中的第39位

## 备注/分析
### 成立与历史
加州州立大学北岭分校成立于1958年，最初名为圣费尔南多谷州立学院（San Fernando Valley State College）。1972年，学校更名为加州州立大学北岭分校。该校是加州州立大学系统内的一个重要成员，以其计算机工程和土木工程专业而闻名。

### 学术成就
加州州立大学北岭分校在2021年《美国新闻与世界报道》的“西部地区大学”排名中位列第39位，这表明该校在学术研究和教学质量方面具有较高的声誉。

### 学生人数与多样性
截至2013年秋季，该校共有学生38,310人，其中本科生33,771人，研究生4,912人。这表明该校吸引了大量的学生，并提供了丰富的学术资源。

### 校园设施与资源
加州州立大学北岭分校拥有353英亩的校区，提供丰富的校园设施和资源。学校致力于为学生提供良好的学习环境和学术支持。

### 校长与教师
Dianne F. Harrison是该校的校长，教师人数为1,700人。这表明该校拥有一支强大的教师队伍，为学生提供高质量的学术指导。

### 资金状况
截至2014年，该校的捐赠基金为8750万美元。这表明该校在资金方面相对充裕，能够支持其学术研究和校园建设。
