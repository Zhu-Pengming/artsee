# 加州州立大学北岭分校（CSUN）艺术学院 Wiki - 决策型版本

## TL;DR
加州州立大学北岭分校（CSUN）艺术学院以其亲民学费、洛杉矶地理位置和跨学科学习氛围而著称。该学院提供多样化的艺术专业，适合寻求性价比高、实习就业机会丰富的艺术教育的学生。以下是对CSUN艺术学院的详细分析，包括学校概况、专业设置、申请要求、校园设施、学生体验以及就业前景。

## 关键声明
### 学校概况
- CSUN艺术学院隶属于Mike Curb College of Arts, Media, and Communication，提供本科和研究生学位。
- 学院强调通识教育与专业艺术训练的平衡，并已获得NASAD认证。

### 专业设置
- 提供动画、艺术教育、艺术史、陶艺、传播设计、绘画、插画、油画、摄影/视频、版画和雕塑等11个方向。
- 本科生无需提交作品集，研究生需提供作品集。

### 申请要求
- 本科生需高中毕业，GPA 3.0/4.0以上，IELTS 6.0 / TOEFL 61。
- 研究生需艺术学士学位或同等学历，GPA 3.0以上，IELTS 6.5 / TOEFL iBT 79。
- 研究生需提供作品集、目的陈述、简历/履历、两封推荐信和官方成绩单。

### 学费与奖学金
- 本科学费$7,458 (州内) / $20,058 (州外/国际生)。
- 研究生学费$14,068 (M.A.) / $16,542 (M.F.A.)。
- 提供多种奖学金，但国际学生申请难度较大。

### 校园设施与创作资源
- 拥有现代化的教学楼和工作室空间，配备行业标准软件和设备。
- CSUN Art Galleries定期举办展览，为学生提供作品展示机会。
- 研究生可获得个人工作室空间。

### 学生体验
- 学生对课程体验的评价差异较大，有学生认为项目富有挑战和成就感，但也有个别学生对教授评分非常低。
- 有学生反馈部分教授非常善于引导学生，帮助建立作品集并给予职业发展建议。
- 学校强调动画专业是其优势专业之一，与梦工厂等有实习和就业合作。

### 就业前景
- 毕业生可在设计、广告、媒体、出版、游戏、画廊和博物馆等行业工作。
- 动画专业与包括梦工厂在内的知名工作室有合作关系，为学生提供关键实习和就业机会。
