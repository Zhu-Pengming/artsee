# 阿伯丁大学简介

## TL;DR
阿伯丁大学是英国五所古典大学之一，成立于1495年，位于苏格兰阿伯丁市。该校是公立综合类研究型大学，以其在石油地质学、医学、商科和法律等领域的卓越成就而闻名。阿伯丁大学提供丰富的本科和研究生课程，包括自然科学、社会科学、人文艺术和医学等。学校拥有悠久的历史和卓越的学术声誉，是英国顶尖的大学之一。

## 其他信息
- **办学历史**: 阿伯丁大学成立于1495年，由罗马教皇授权建立。1860年，国王学院与马修学院合并，正式称为阿伯丁大学。
- **办学条件**: 阿伯丁大学下设超过390个第一学位课程，100多门授课式硕士和研究生文凭课程以及传统的博士研究项目。
- **招生录取**: 阿伯丁大学提供本科和研究生课程，入学要求包括学术成绩、语言成绩等。
- **社会评价**: 阿伯丁大学是英国最古老的五所大学之一，拥有卓越的学术声誉和良好的社会评价。
- **校园环境**: 阿伯丁大学位于风景怡人的苏格兰东海岸第三大城市阿伯丁市，拥有美丽的校园环境和先进的设施。
- **知名校友**: 阿伯丁大学培养了许多杰出人才，包括诺贝尔奖得主、政治家、科学家、艺术家等。
- **合作交流**: 阿伯丁大学与多个国际机构合作，开展联合培养项目和科研合作，提升了学校的国际影响力。
- **文化传统**: 阿伯丁大学的学生会组织了150多个俱乐部和社团，为学生们提供了丰富的社交活动。
- **学校领导**: 阿伯丁大学的历任领导包括英国前首相丘吉尔、前任港督卫亦信等。
- **校园住宿信息**: 阿伯丁大学提供校内住宿和校园附近的私有住处，为学生提供便利的住宿条件。
- **图书馆**: 阿伯丁大学的图书馆服务分布在四个地点，为学生提供丰富的学习资源。
- **信息技术和计算机**: 阿伯丁大学为学生提供先进的计算机设施和无线网络连接。
- **知名校友**: 阿伯丁大学的知名校友包括詹姆斯·克拉克·麦克斯韦、温斯顿·丘吉尔、弗雷德里克·索迪、杨昌济、伊恩·格雷、安妮公主、何启等。
