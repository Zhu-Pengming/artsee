# 皇家墨尔本理工大学深度调研报告

## TL;DR

- RMIT 在艺术设计领域具有卓越声誉，连续三年位列澳洲第一、全球前30，但计算机科学/工程的研究声誉与就业品牌相对较弱。
- RMIT 是一所应用型大学，学费相对合理，适合追求艺术设计+技术交叉、看重产业实战和本地就业的学生。
- 对于有Meta工作经验和Northeastern ECE硕士背景的学生，RMIT的计算机科学/IT项目性价比偏低，但设计/创意技术方向（如动画、游戏、交互设计）具有竞争力。
- RMIT 的国际学生比例高，提供多种住宿选项，并拥有丰富的校友资源和职业发展支持。
