# RMIT新生校园一日记录：逛欢迎派对、办电话卡交通卡全流程分享

## TL;DR
本文记录了一名RMIT新生的校园一日生活，包括参观欢迎派对、办理学生卡、电话卡和交通卡，以及介绍校园设施、周边环境和学校特色。

## Key Claims
- RMIT位于墨尔本市中心，交通便利，周边配套设施完善。
- 学校欢迎派对活动丰富，特色布置吸引新生。
- 学生卡办理流程需在线上传照片，学校邮寄。
- 澳洲电话卡办理推荐OPTUS套餐，性价比高。
- 澳洲交通卡办理方便，市区存在免费区域。

## Evidence / Methodology

- 视频内容展示校园环境和活动情况。

## RMIT校园设施与周边环境
- RMIT位于墨尔本市中心，交通便利，周边商场、图书馆等配套设施完善。
- 学校建筑风格独特，商科大楼由本校建筑系学生设计。
- 校内商店售卖学校周边商品，价格合理。

## 学生卡办理流程
- 学生需登录个人学生账户，上传照片，填写家庭住址。
- 学校在一周后将学生卡邮寄到住址。

## 欢迎派对活动
- 派对邀请多支乐队演出，设置学生摊位和商家宣传摊位。
- RMIT在派对现场设置可移动小型摩天轮，特色布置吸引新生。

## 澳洲电话卡办理
- 学生推荐OPTUS套餐，性价比高，包含80G流量和不限时免费国际通话。

## 澳洲交通卡办理
- 学生可在任意地铁站办理，需缴纳6澳元押金，可按需充值余额。

- 学校采用六等级制评分，GPA只按等级计算。
