# 纽约州立大学布法罗分校（艺术系）

## TL;DR
纽约州立大学布法罗分校（University at Buffalo, SUNY，简称UB）是一所位于美国纽约州布法罗的公立研究型大学，以其学术研究和教学质量著称。该校拥有多个学院和专业，包括艺术系，并以其丰富的历史、卓越的学术成就和多样化的学生群体而闻名。

## 关键声明
- **历史与地位**：UB是纽约州立大学系统中规模最大、研究经费最高、综合性最强的公立旗舰大学，被誉为“公立常春藤”之一。
- **学术研究**：UB被卡内基基金会归类为特高研究型大学（R1级别），在工程与计算机科学领域积淀深厚。
- **学术成就**：学校曾涌现1位美国总统、4位诺贝尔奖、5位普利策奖及3位菲尔兹奖得主。
- **学院与专业**：UB拥有13个学院，开设400多个学士、硕士和博士专业，其中艺术系是其中之一。
- **国际学生**：近11%的UB学生有机会前往其他国家一流大学学习，UB还是美国为数不多的开设有孔子学院的大学之一。
- **校园环境**：UB位于纽约州布法罗市，由南校区、北校区以及市内校区三个校区组成，校园面积1,350英亩。
- **知名校友**：UB的校友及教职人员成就杰出，包括一任美国总统，多位宇航员、公司总裁以及普利策奖、美国国家科学奖、国家技术创新奖得主。
