# 纽约州立大学布法罗分校艺术系Wiki - 决策型版本

## TL;DR
纽约州立大学布法罗分校艺术系提供融合当代艺术、科技与批判性理论的跨学科艺术教育。该系在US News排名中位列第73名，提供本科和研究生项目，包括绘画、平面设计、摄影、版画媒体、雕塑等。学费和生活费相对较低，提供全额学费奖学金和助教职位。校园设施包括24小时实验室、个人工作室和艺术画廊。学生体验和就业数据主要来自UGC，存在一定偏差。
