# 明尼苏达大学双城分校简介

## TL;DR
明尼苏达大学双城分校是一所位于美国明尼苏达州的双城校区，拥有悠久的历史和卓越的学术声誉。学校提供370多个专业，涵盖多个学科领域，并以其在工程、心理、经济等领域的优势而闻名。本文将详细介绍该校的办学历史、院系设置、专业设置、招生录取、学术研究、校园环境、住宿餐饮、交通设施、校园活动、知名校友等方面的信息。

## 关键信息

### 办学历史
- **创建时间**：1851年
- **办学性质**：公立大学
- **学校类别**：综合类研究型大学
- **学校特色**：公立常春藤，21世纪学术联盟成员，十大联盟成员

### 院系设置
- 学校由20所院校组成，提供370多个专业。
- 主要学院包括：卡尔森管理学院、文理学院、理工学院等。

### 专业设置
- 本科专业涵盖金融、会计、应用经济学、国际商务、统计学、数学、建筑学、土木工程、化学、化学工程、计算机工程、生物化学、生物学、药物技术、生物系统工程、航空航天工程和机械学、服装设计、通讯、计算机科学、施工管理、电机工程、企业管理、食品科学、人力资源和劳资关系、信息技术、室内装饰、原料科学和工程、数学、机械工程、音乐、艺术、物理、心理学、数据科学、电机工程、环境工程、地球工程、工业和系统工程、材料科学与工程、机械工程、天体物理学、生物制品与生物系统工程、土木工程等。
- 研究生专业涵盖经济学、土木工程、航空航天工程、金融数学、建筑、景观建筑、艺术、生物科学、生物工程、生物统计学、化学工程、材料科学、化学物理、化学、医学化学、通讯、计算机科学、电机工程、工业工程、内部装饰设计、新闻学、技术管理、大众传媒、数学、机械工程、微生物工程、音乐、神经系统学、营养学、制药学、物理学、心理学、软件工程、统计学等。

### 招生录取
- 本科申请截止日期：建议申请截止时间为12月16日
- 转学申请截止日期：根据学院和专业有所不同
- 留学费用：本科学杂费/年（州内学生 $14,693，州外学生$30,371）；研究生学费/年（州内学生$18,492，州外学生$27,648）
- 奖学金：向国际学生提供奖学金
- 申请材料：高中成绩单、托福成绩单、申请费、财力证明表、申请表
- 入学要求：本科托福ibt79或雅思6.5，MELAB80 GPA3.4；硕士托福ibt90（不同学院要求不同）或雅思7，及GRE或GMAT

### 学术研究
- 学校在各类全球大学排名中几乎均列位前50名。
- 经济、机械、化学、化工、数学、建筑学、药学和教育等专业世界驰名。
- 在过去的5年里，共获239个科研专利，并且帮助建立了35个公司发展生产新产品。

### 校园环境
- 地理位置：美国明尼苏达州明尼阿波利斯市和圣保罗市
- 校园面积：1,204（英亩）
- 学校共有5个校区，其中最好最著名的便是Twins Cities （双城）校区。

### 住宿餐饮
- 学校提供12栋宿舍楼可供学生选择，包括普通宿舍和套房类宿舍。
- 学校周边地区也有校外的住宿选择。
- 学校中有超过30个食堂、餐厅、咖啡店和方便的小商店，提供10种meal plan。

### 交通
- 最近的机场：明尼阿波利斯 Minneapolis 10英里
- 最近的火车站：圣保罗St. Paul 5英里
- 校内设有公共交通，包括公交线路和轻轨。

### 校园活动
- 明尼苏达大学有几百个学生组织，包括环保、兄弟会、艺术、学术、政治、宗教等各个方面。

### 知名校友
- 学校培养了25位诺贝尔奖得主，包括物理学家、化学家、经济学家、文学家等。
