# 巴塞罗那大学美术学院决策型Wiki

## TL;DR
巴塞罗那大学美术学院(Facultat de Belles Arts, Universitat de Barcelona)是西班牙历史最悠久、规模最大的公立大学之一中的美术学院，也是加泰罗尼亚地区唯一的美术学院。该学院提供本科及多个官方硕士项目，以公立低学费(本科约€4,246/年，硕士约€1,162-4,920/年)为特色，但西班牙语授课为主，英语授课比例极低。学院强调实验性创作与跨学科研究，依托巴塞罗那丰富的当代艺术生态，但在传统studio技能训练方面弱于欧洲独立艺术学院。适合预算有限、已有西班牙语基础、对当代艺术理论感兴趣的学生，不适合追求纯英语环境或高额奖学金的学生。

### 学校概况与定位

巴塞罗那大学美术学院是西班牙公立综合大学系统中的艺术教育机构，具有独特的区域垄断地位和特定的教育定位。

**历史与规模**：
- 巴塞罗那大学成立于1450年，是西班牙历史最悠久、规模最大的公立大学之一
- 美术学院(Facultat de Belles Arts)位于巴塞罗那Diagonal校区
- 作为加泰罗尼亚地区唯一的美术学院，具有区域垄断性优势
- 学院提供本科(Belles Arts)及多个官方硕士项目，涵盖当代艺术创作、艺术研究、声音艺术、视觉艺术教育、设计艺术指导等方向

**教育特色**：
- 强调实验性创作与跨学科研究
- 拥有工作坊和技术实验室支持创作需求
- 依托巴塞罗那丰富的当代艺术生态(MACBA、CCCB、毕加索博物馆、众多画廊)
- 在理论研究和社会介入方面具有鲜明特色
- 传统studio技能训练相对欧洲独立艺术学院较弱

**语言环境**：
- 授课以西班牙语和加泰罗尼亚语为主
- 本科课程中加泰罗尼亚语占76%，西班牙语占24%，英语授课比例<1%
- 硕士项目主要为西班牙语授课，仅部分设计硕士有英语方向
- 校园和当地生活中加泰罗尼亚语无处不在，虽授课用西班牙语，但行政、标识、本地交流多用加泰罗尼亚语

### 适合人群与不适合人群

**适合申请者**：
