# 巴塞罗那大学美术学院 Wiki（决策型）

## TL;DR
巴塞罗那大学美术学院（Facultat de Belles Arts, UB）是西班牙公立美术教育的旗舰之一，提供从本科到博士的多种艺术课程。学院位于巴塞罗那市区，以其文化遗产保护与修复专业而闻名。虽然学院在英语艺术圈的品牌知名度低于RISD、UAL等，但其在加泰罗尼亚和西班牙国内享有盛誉。本文基于AI生成的Wiki内容，分析了学院的课程设置、排名、适合人群、申请要求、学费和生活费等关键信息。

## 关键 Claims

### 1. 学院简介
- 巴塞罗那大学美术学院（Facultat de Belles Arts, UB）隶属于巴塞罗那大学，前身是1775年成立的免费设计学校，1849年并入西班牙省级美术学院体系。
- 学院提供官方学位包括本科（Grau）、研究生硕士（Màster universitari）和博士（Doctorat），涵盖美术、文化遗产保护-修复、设计等多个领域。
- 学院位于巴塞罗那市区，是加泰罗尼亚地区历史最深、规模最大的公立美术高等教育机构。

### 2. 学校层级与排名
- 巴塞罗那大学（UB）在2026 QS世界大学排名中位列第160，是西班牙排名第一的大学。
- 美术学院本身是西班牙公立美术教育的旗舰之一，与马德里康普顿斯美术、瓦伦西亚理工美术并列为西班牙艺术留学Top 3。
- 学院在QS Art & Design学科排名中位于201+（top 250段），明显低于专业艺术学院（如RCA、UAL、RISD）。

### 3. 适合人群
- 想在欧盟体系学纯艺术（painting / sculpture / printmaking / drawing）的人
- 预算敏感：想读公立学校而非英美私立艺术院校
- 看重欧洲长期居留与申根流动性（西班牙学生签可游欧盟）
- 想读艺术理论 / 艺术史 / 哲学跨学科方向的人
- 想读文化遗产保护-修复方向
- 想读艺术教育 / 视觉艺术教育方向的人
- 申请研究型博士（Doctorat），想做art practice research的人

### 4. 不适合人群
- 一句西语 / 加泰罗尼亚语不会但希望本科英语授课的人
- 想要纯艺术学院studio-only氛围的人
- 目标是英美tech / 工业设计 / 交互设计就业的人
- 想冲国际艺术品牌（如RCA、UAL、RISD、Parsons）光环的人
- 需要高额艺术MFA funding的人
- 偏好Fashion / Industrial Design工业级资源

### 5. 课程设置
- 本科课程包括美术、文化遗产保护-修复、设计等。
- 硕士课程涵盖艺术创作与研究、城市设计、字体设计、文化遗产保护与修复、视觉艺术、艺术教育等。
- 博士课程培养学生在艺术生产与设计领域的高级研究能力。

### 6. 语言要求
- 学院教学语言比例：约60%课程加泰罗尼亚语，35%课程西班牙语，少于5%课程其他语言。
- 国际生实际进入本科课堂需要B2西语或加泰罗尼亚语。
- 部分硕士项目（如MBDesign的Contemporary Design / Innovation方向）英语授课。

### 7. 作品集要求
- 本科Belles Arts与Disseny在国际生申请通道下需提交作品集。
- 具体规格每年招生窗口公布，未公开统一规格。
- 中文中介经验：常见要求15-25件作品PDF；涵盖素描、色彩、创作三类；偏重传统美术基本功。

### 8. 申请要求
- 本科国际生通道：通过UNEDasiss验证学历，UB在线申请，提交相关材料。
- 硕士国际生通道：EHEA内学位直接申请，EHEA外学位需homologación或申请UB内部学位等同性核查。

### 9. 学费与奖学金
- 学校级一般水平：国际学生平均年学费约€4,900。
- 硕士项目60 ECTS：标准入学费区间€2,500-€3,300。
- 西班牙公立大学没有像美国MFA那样的全奖体系，但部分项目有竞争性奖学金。

### 10. 竞争激烈程度
- 学校级acceptance rate约69%（2025） / 75%（部分来源）。
- 热门项目：MBDesign英语方向、Producció i Recerca Artística、Conservació-Restauració。
- 冷门项目：传统纯艺术方向（painting / engraving）通常竞争更小。

### 11. 校园设施与创作资源
- 学院位于巴塞罗那市中心/Pedralbes校区附近，拥有丰富的创作资源。
- 学院拥有CRAI Library、Modern Languages School、Mobility / 交换办公室、毕业展、校内画廊、工作室等设施。
- 巴塞罗那本身的城市艺术生态是UB美术学院最大隐性资源。
