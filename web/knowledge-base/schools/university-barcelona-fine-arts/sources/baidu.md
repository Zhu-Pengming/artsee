# 巴塞罗那大学介绍

## TL;DR
巴塞罗那大学（Universidad de Barcelona，简称UB）是一所位于西班牙巴塞罗那的著名公立大学，拥有悠久的历史和卓越的学术成就。学校提供丰富的本科和研究生课程，涵盖多个学科领域，并以其艺术与设计、医学、人文、理学及政商传统而闻名。巴塞罗那大学在全球多个大学排名中位列前列，拥有来自超过120个国家的学生，并以其高质量的教育和研究而受到国际认可。

## 关键陈述

- **学术成就**：学校在多个学科领域均取得卓越成就，在全球多个大学排名中位列前列。

- **院系与专业**：学校设有多个学院和研究所，提供丰富的本科和研究生课程。
- **教学语言**：本科课程通常使用加泰罗尼亚语授课，而研究生课程则提供多种语言选项。
- **学费与奖学金**：学费根据课程性质和学生的居留身份有所不同，学校提供多种奖学金机会。
