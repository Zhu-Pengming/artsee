# 突尼斯高等美术学院 - 决策型版本

## TL;DR
突尼斯高等美术学院（ISBAT）是突尼斯最古老、规模最大的公立高等艺术院校，拥有悠久的历史和深厚的艺术底蕴。然而，由于其国际知名度有限，对于国际学生（尤其是中国学生）而言，选择ISBAT可能面临信息不透明、就业前景不明朗等挑战。

## 关键声明

### 学校概况
- ISBAT是突尼斯最古老、规模最大的公立高等艺术院校，隶属于突尼斯大学。
- 学校拥有艺术、设计两个完整院系及博士学院，提供从学士到博士的完整教育体系。
- 学校在突尼斯及马格里布地区享有盛誉，但在全球艺术院校体系中影响力有限。

### 项目与专业
- ISBAT提供艺术与设计两大方向的完整本硕博体系，包括绘画、雕塑、摄影、陶瓷、视觉艺术、艺术理论、产品设计、空间设计等。
- 学校与法国多所大学合作，提供跨学科研究机会。

### 申请要求
- 突尼斯申请人通过国家统一高校招生系统申请，国际申请人需提供相当于Baccalauréat的高中毕业资格，并可能需要参加入学考试。
- 硕士申请者需具备相关专业背景，建议法语B2水平。
- 作品集可能作为补充评估，但具体要求未明确。

### 学费与奖学金
- 学校对本国学生学费低廉，国际学生学费需直接联系校方确认。
- 未找到官方奖学金信息。

### 竞争激烈程度
- 国际学生数量极少，竞争程度低。
- 对本国学生录取门槛较高，但未找到具体录取率数据。

### 校园设施与创作资源
- 学校拥有图书馆、阶梯教室、教室、学生俱乐部空间、餐饮、材料图书馆等设施。
- 提供丰富的创作资源，包括各种艺术材料。

### 学生体验与就业
- 未找到中国学生就读体验分享。
- 未找到官方毕业生就业去向数据，校方宣称毕业生就业能力较强。

### 周边艺术资源
- 学校位于突尼斯市，周边拥有丰富的艺术资源，包括Belvédère艺术馆、突尼斯麦地那等。

## 备注/分析

### 学校优势
- **历史悠久**：ISBAT成立于1923年，是突尼斯第一所高等艺术院校，拥有丰富的艺术底蕴和历史积淀。
- **规模庞大**：作为突尼斯最大的艺术教育机构，ISBAT拥有丰富的教学资源和创作平台。
- **项目齐全**：提供从学士到博士的完整教育体系，满足不同层次学生的需求。
- **合作网络**：与法国多所大学合作，为学生提供跨学科研究机会。

### 学校劣势
- **国际知名度有限**：ISBAT在国际艺术院校体系中影响力有限，对于国际学生而言，选择ISBAT可能面临信息不透明、就业前景不明朗等挑战。
- **语言障碍**：主要教学语言为法语，对于非法语学生而言，语言可能成为一大障碍。
- **就业前景不明朗**：未找到官方毕业生就业去向数据，对于国际学生而言，就业前景不明朗。
- **国际化程度低**：国际学生数量极少，国际化程度低。

### 对中国学生的启示
- 对于中国学生而言，选择ISBAT可能面临信息不透明、就业前景不明朗等挑战。
- 建议在申请前详细了解学校情况，并与在校生或校友进行交流，以获取更多一手信息。
- 考虑自身语言能力和就业目标，选择适合自己的留学目的地和院校。

## 其他信息

### 学校历史
- 1923年，ISBAT以“艺术教育中心”名义在突尼斯麦地那区Dribat Ben Abdallah创立。
- 1930年，更名为“突尼斯美术学院”。
- 1949年，多名校友创立现代突尼斯艺术史上的重要流派——“突尼斯画派”。
- 2008年，参与突尼斯大学建校50周年庆典。
- 2010年，举办以纺织艺术为主题的学生作品展，主题为“编织”，作品在Belvédère艺术馆展出。
- 2018年，举行95周年校庆，举办设计、装置与绘画雕塑等艺术联展。

- 突尼斯公立艺术院校排名第1位，突尼斯所有大学中综合排名第23452位，得分为12.06。

### 学校设施
- 图书馆、阶梯教室、教室、学生俱乐部空间、餐饮、材料图书馆等。

### 学校合作
- 与法国多所大学保持学术合作关系，提供硕士和博士层面的跨学科研究机会。

### 学校荣誉
- 校友创立了现代突尼斯艺术史上最重要的画派之一——“突尼斯画派”。
