## TL;DR
萨凡纳艺术与设计学院（SCAD）是美国顶尖的独立艺术与设计学院之一，以其专业广度和行业接驳著称。本文详细分析了SCAD的学校概况、排名、适合人群、项目数据、申请要求、学费奖学金、竞争激烈程度、校园设施、学生体验、校友故事、就业去向以及周边艺术资源等信息。

## 关键声明
- SCAD是美国"实战型"独立艺术与设计学院的代表，专业广度全美最大，行业就业导向极强。

- SCAD的录取率约为83%，在中国学生圈口碑两极。

## 学校概况
SCAD成立于1978年，由Paula Wallace、Richard Rowan、May Poetter、Paul Poetter在乔治亚州Savannah创立。学校规模上是全美最大的纯艺术与设计院校之一，拥有约18,500名在校生，来自100+个国家，教职工约700名。

### 学校特点
- **专业广度**: SCAD提供100+个本研项目，覆盖艺术、设计、动画、建筑、影视、时尚、表演艺术等领域。
- **行业接驳**: SCAD的项目大量配合产业实际工种设立，如UX、Game Dev、Themed Entertainment、Service Design、Luxury Brand Management、Themed Entertainment Design等。
- **quarter学制**: 学校采用quarter学制，每年4个quarter，每门课5学分，与多数美国学校的semester不同。
- **地理位置**: SCAD主校区位于Savannah，同时在Atlanta、Lacoste(France)、SCADnow(线上)设有校区。

## 语言、作品集与申请要求
### 语言要求
- TOEFL iBT: 85（每个 section ≥ 20）
- IELTS: 6.5 总分，Reading + Writing 各 ≥ 6.5
- Duolingo: 120+
- SAT: Reading ≥ 550
- ACT: Reading & Writing ≥ 22
- 豁免条件: 全英授课本/研学位 或 高中全程英语授课
- 替代方案: SCAD Language Studio（ESL 内部学习）

### 作品集要求
- 本科首次申请 / 转学: 不强制，但强烈建议交
- 研究生: 必交（必须 major-specific）
- 提交平台: SlideRoom
- 通用要求: 作品集需呈现「问题识别 → 替代方案 → 解决方案 → 结果」的设计过程；研究生必须 major-specific

### 申请要求
- 滚动招生，无统一deadline
- 国际生建议入学前至少30天提交完整申请
- 录取通常2-4周出结果
- 国际申请费 ~$100
- 接受Common Application；Test-optional

## 住宿、交通与城市生活
### Savannah（主校区）
- 城市规模: 沿海小城，约15万+人口；SCAD ~15,000学生占很大比例
- 城市氛围: 历史名城（antebellum建筑、橡树、西班牙苔藓、马车）、艺术 / 旅游业为主、节奏慢
- 气候: 亚热带，**夏季湿热**
- 安全: Safety index ~62（amberstudent.com）；但College Confidential学生帖提到Savannah "比Atlanta更有犯罪问题"
- 住宿费用: On-campus ~$10,719/yr；off-campus普遍更便宜（校友建议）
- 公共交通: 校区分散，校内有SCAD班车SCAD Bus；公共交通有限，**有车更方便**
- 中国超市 / 中餐: **资源较少**（小城特性）；亚特兰大或Charleston是补给点

### Atlanta
- 城市规模: 大都市（metro 600万+）、美国南方科技 / 媒体中心
- 工作机会: 时尚 / 影视 / 设计 / 创业资源；Delta、CocaCola、可口可乐、Tyler Perry Studios 在地
- 住宿费用: 高于Savannah，但仍低于纽约 / 旧金山
- 中文资源: 比Savannah多得多（中国超市 / 中餐 / 中国教会）

### 城市选择建议
- 想要**专心做项目、不被城市分散注意力** → Savannah
- 想要**实习、行业接触、社交机会** → Atlanta
- 但**学位是同一个 SCAD**，学位本身不会标 "Atlanta / Savannah" 来歧视任何一边
