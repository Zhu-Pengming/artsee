# 开罗美术学院

## TL;DR
开罗美术学院，成立于1908年，是埃及历史悠久的艺术教育机构。学院由优素福·卡迈勒创立，提供绘画、装饰、雕塑、建筑和书法等课程。学院在埃及艺术史上具有重要地位，曾举办首次埃及艺术家展览，并受到埃及教育部接管。

## 关键声明
- **成立时间**: 1908年
- **创立者**: 优素福·卡迈勒
- **学生数量**: 创立初期招收150多名学生
- **教育对象**: 所有国籍和宗教信仰的学生均可免费接受教育
- **课程设置**: 绘画、装饰、雕塑、建筑和书法
- **首任院长**: 纪尧姆·拉普拉涅（法国雕塑家）
- **首次展览**: 1911年
- **接管机构**: 埃及教育部（1928年）
