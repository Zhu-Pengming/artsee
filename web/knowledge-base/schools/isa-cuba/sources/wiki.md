# 古巴国家艺术学校

## TL;DR
古巴国家艺术学校（Escuelas Nacionales de Arte）是古巴革命期间的重要建筑成就，由三位建筑师Ricardo Porro、Roberto Gottardi 和 Vittorio Garatti设计。学校包含五所艺术学校：现代舞学院、造型艺术学院、戏剧艺术学院、音乐学院和芭蕾舞学院。然而，由于政治和意识形态原因，学校在1965年建设停滞，部分建筑被遗弃。

## 详细分析
### 建筑成就
古巴国家艺术学校是古巴革命期间的重要建筑成就，由三位建筑师Ricardo Porro、Roberto Gottardi 和 Vittorio Garatti设计。学校的设计与当时国际风格背道而驰，强调与自然景观的结合和当地材料的使用。学校包含五所艺术学校，每所学校都有其独特的建筑和设计理念。

### 建筑设计
学校的设计体现了对现代主义的批判和对古巴革命形象的重建。建筑师们使用当地生产的砖和赤陶瓦，以及具有有机形式潜力的加泰罗尼亚拱顶。这种设计理念强调了与自然景观的结合，以及使用当地材料和工艺。

### 五所学校
学校包含五所艺术学校，每所学校都有其独特的建筑和设计理念。
- **现代舞学院**：由Ricardo Porro设计，其设计理念是将现代舞学校的计划设想为一块玻璃，被猛烈地砸碎并碎裂成移动的碎片，象征着革命对旧秩序的暴力推翻。
- **造型艺术学院**：由Ricardo Porro设计，其概念旨在唤起一个典型的非洲村庄，创造一个由街道、建筑和开放空间组成的有机城市综合体。
- **戏剧艺术学院**：由Roberto Gottardi设计，其设计理念是城市化，围绕中央广场圆形剧场组织成一个紧凑的轴向蜂窝平面图。
- **音乐学院**：由Vittorio Garatti设计，其设计理念是一条330米长的蛇形丝带，嵌入并穿过靠近河流的景观轮廓。
- **芭蕾舞学校**：由Vittorio Garatti设计，其设计理念是从高尔夫球场的峡谷顶部，可以俯瞰坐落在下降峡谷中的芭蕾舞学校综合体。

### 政治影响
由于政治和意识形态原因，学校在1965年建设停滞，部分建筑被遗弃。这反映了古巴政治环境的变化，以及学校在政治上的敏感性。

## 其他信息
- **建设时间**：学校建设始于1961年，但由于政治和意识形态原因，在1965年建设停滞。
- **建筑师**：Ricardo Porro、Roberto Gottardi 和 Vittorio Garatti
- **地点**：哈瓦那，古巴
- **现状**：部分建筑被遗弃，但仍有部分建筑在使用中。
