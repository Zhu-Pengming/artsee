# 古巴高等艺术学院（Universidad de las Artes）概述

## TL;DR
古巴高等艺术学院（现称古巴艺术大学）是古巴最重要的艺术教育机构之一，成立于1976年，提供音乐、视觉艺术、戏剧、舞蹈、视听媒体艺术和文化遗产保护等领域的本科和研究生教育。学院拥有强大的师资力量，与多个国家艺术院校有交流协议，培养了许多在国际上知名的艺术家。
