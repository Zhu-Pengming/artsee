# 开罗大学简介

## TL;DR
开罗大学是埃及和阿拉伯世界最古老的高等教育机构之一，始建于1908年，位于埃及首都开罗。作为埃及最大的高校，该校拥有丰富的历史和文化底蕴，在继承和发扬古埃及文化、阿拉伯伊斯兰文化方面具有重要作用。开罗大学在多个学科领域均有突出表现，拥有众多知名校友，包括联合国秘书长、诺贝尔奖获得者等。

## 关键声明
- **历史与地位**：开罗大学是埃及和阿拉伯世界最古老的高等教育机构之一，始建于1908年，位于埃及首都开罗。
- **规模与结构**：开罗大学是埃及最大的高校，拥有3个分校、26个学院、177个机构和中心、5个学术部门，1.6万名教师及26万在校学生。
- **学科特色**：开罗大学在医学类、文学类和法学类学科领域具有显著优势，同时商学类、工学类和农学类科目也举足轻重。
- **国际合作**：开罗大学积极参与国际合作，与多所世界知名高校建立了合作关系。
- **学校排名**：2025年U.S. News世界大学排名位列第271名，QS世界大学排名第350名。
- **知名校友**：开罗大学培养出众多知名校友，包括联合国秘书长、诺贝尔奖获得者等。
