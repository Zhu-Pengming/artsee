# 韩国国立艺术大学

## TL;DR
韩国国立艺术大学（韩国艺术综合学校）是韩国唯一一所国立艺术大学，成立于1993年，位于首尔。学校提供学士和硕士学位课程，涵盖音乐、舞蹈、戏剧、电影、电视、动画、美术、设计、建筑以及韩国传统艺术等所有艺术门类。学校设有六个学院，共26个系，拥有众多杰出校友和成就。

## 关键声明
- **成立时间与背景**：韩国国立艺术大学成立于1993年，由韩国文化体育观光部创办，旨在培养顶尖艺术家。
- **学院与系所**：学校设有六个学院，包括音乐学院、戏剧学院、电影电视与多媒体学院、舞蹈学院、视觉艺术学院和韩国传统艺术学院，共26个系。
- **学生与教职工**：截至2023年9月30日，学校共有2636名本科生和1052名硕士生，136名教授和447名教职工。
- **学术成就**：学校培养出多位在国际上获得认可的艺术家，包括获得世界级钢琴比赛冠军的亚洲音乐家金善旭、赢得Long-Thibaud国际小提琴比赛的申贤秀等。
- **校园与设施**：学校由三个校区组成，包括石馆校区、瑞草校区和大学路校区，提供丰富的教学资源和设施。

## 学院与系所
### 音乐学院
- **声乐系**：培养具有专业声乐技巧和音乐素养的歌唱家。
- **歌剧系**：专注于歌剧表演和创作。
- **艺术歌曲与清唱剧系**：培养艺术歌曲和清唱剧表演人才。
- **器乐系**：涵盖各种乐器演奏技巧。
- **作曲系**：培养作曲家和音乐理论家。
- **音乐技术系**：研究音乐科技和音乐制作。
- **指挥系**：培养指挥家和音乐编导。
- **音乐学系**：研究音乐史和音乐理论。

### 戏剧学院
- **表演系**：培养戏剧表演人才。
- **导演系**：培养戏剧导演人才。
- **剧本创作系**：培养戏剧剧本创作人才。
- **舞台设计系**：培养戏剧舞台设计人才。
- **戏剧研究系**：研究戏剧理论和历史。

### 电影电视与多媒体学院
- **电影制作系**：培养电影制作人才。
- **电影研究系**：研究电影理论和历史。
- **多媒体系**：培养多媒体制作人才。
- **动画系**：培养动画制作人才。
- **广播系**：培养广播制作人才。

### 舞蹈学院
- **舞蹈表演系**：培养舞蹈表演人才。
- **编舞系**：培养舞蹈编导人才。
- **舞蹈理论系**：研究舞蹈理论和历史。
- **艺术管理系**：培养舞蹈艺术管理人才。

### 视觉艺术学院
- **美术系**：培养美术创作人才。
- **设计系**：培养设计人才，涵盖平面设计、工业设计、室内设计等。
- **建筑系**：培养建筑人才。
- **艺术理论系**：研究艺术理论和历史。

### 韩国传统艺术学院
- **传统艺术理论系**：研究韩国传统艺术理论和历史。
- **传统音乐系**：培养韩国传统音乐表演人才。
- **传统舞蹈系**：培养韩国传统舞蹈表演人才。
- **传统民间戏剧系**：研究韩国传统民间戏剧理论和历史。

## 校园与设施
韩国国立艺术大学由三个校区组成，分别是石馆校区、瑞草校区和大学路校区。各校区拥有丰富的教学资源和设施，包括教室、工作室、图书馆、表演厅等。

- **石馆校区**：位于首尔市中心，是学校最早的校区，拥有多个艺术工作室和表演厅。
- **瑞草校区**：位于首尔市南部，是学校的主要校区，拥有多个学院和系所的办公地点，以及丰富的教学设施。
- **大学路校区**：位于首尔市北部，是学校的新校区，拥有现代化的教学楼和设施。

## 杰出校友
韩国国立艺术大学培养出许多在国际上获得认可的艺术家，以下是一些杰出校友的例子：

- **金善旭**：获得世界三大钢琴比赛之一——利兹国际钢琴比赛冠军的亚洲音乐家。
- **申贤秀**：赢得Long-Thibaud国际小提琴比赛的冠军。
- **河恩智**：纽约国际芭蕾舞比赛一等奖获得者，加入芬兰国家芭蕾舞团担任首席舞者。
- **罗泓轸**：韩国电影《追击者》导演，该片获得多项国际奖项。
- **林云灿**：获得范·克莱本国际钢琴比赛奖，后转学到美国马萨诸塞州波士顿的新英格兰音乐学院学习。
