# 巴黎国立高等美术学院（ENSBA）概况

## TL;DR
巴黎国立高等美术学院（ENSBA）是法国最古老、最负盛名的美术学府之一，拥有深厚的历史底蕴、丰富的资源和国际声誉。学院强调工作室教学和创作实践，培养具有独立创作能力和文化底蕴的艺术家。本文详细介绍了ENSBA的建校历史、校园与建筑、开放与参观、使命与特色、学术项目、入学与申请、教师与研究、学生生活与资源、校友与影响、最新动态以及与巴黎其他美院的对比。

## 关键声明

### 建校与地位
- ENSBA成立于1648年，是法国文化部直属国立高等艺术学院，为巴黎高科（PSL）大学成员。
- 学院是巴黎最古老、最负盛名的美术学府之一，肩负培养高水平艺术创作人才的使命。

### 校园与建筑
- 校园分布于巴黎左岸圣日耳曼德佩和郊区圣图安两处。
- 巴黎校区拥有画室、礼堂、图书馆及展厅；圣图安校区设有雕塑、模塑、铸造、陶瓷等实验车间。

### 开放与参观
- 学院建筑为历史遗址，仅在特定时段免费对外开放。
- 日常教学场所不对外开放，公众参观须关注官网公告。

### 使命与特色
- ENSBA强调工作室教学和创作实践，在名师指导下，学生在绘画、雕塑等专业工作室创作。
- 学院传统与当代并重，致力于培养具有独立创作能力和文化底蕴的艺术家。

### 学术项目
- 本硕学制：本科3年，硕士5年。
- 学分与学位：3年本科后设答辩/作品陈述；取得本科证书后进入第4年；两年后取得DNSAP硕士学位。
- 博士项目：参与PSL大学“SACRe”跨校博士项目，强调“创作+研究”融合。

### 入学与申请
- 申请流程：法国高等教育招生平台Parcoursup统一报名。
- 语言要求：法语至少B2水平。
- 作品集要求：需提交PDF格式的艺术作品集或作品演示视频。

### 教师与研究
- 师资力量：由当代知名艺术家、学者组成。
- 研究团队与方向：强调“创作即研究”的理念，关注交叉学科议题。

### 学生生活与资源
- 住宿与生活：校内学生宿舍有限，校外住宅需自理。
- 学生组织与社团：ENSBA学生会及各画室社团活跃。
- 展览与实践机会：学院全年有各类展览，学校开放空间定期举办公开展览。

### 校友与影响
- 著名校友：徐悲鸿、林风眠、亨利·马蒂斯等。
- 校友网络：遍布全球艺术机构和高校，拥有广泛人脉。
- 职业路径：毕业生多担任艺术家、教师或独立艺术顾问。

### 最新动态
- 与圣图安市政府和Audencia商学院签署城市艺术教育合作协议。
- 参与PSL大学联合博士（SACRe）招生。
- 在QS世界大学艺术设计专业排名中，ENSBA保持稳定上升。

### 与巴黎其他美院对比
- ENSBA偏重传统美术创作，录取要求严苛。
- 巴黎装饰艺术学院（EnsAD）专注设计领域，课程多样。
- 巴黎高等美术学院（ENSAPC巴黎-塞纳）注重新媒体和数字艺术。
