# 马萨诸塞大学阿默斯特分校简介

## TL;DR
马萨诸塞大学阿默斯特分校（UMass Amherst）是一所位于美国马萨诸塞州阿默斯特镇的公立研究型大学，隶属于麻省大学系统。该校成立于1863年，拥有悠久的历史和卓越的学术声誉。UMass Amherst提供广泛的本科和研究生课程，尤其在计算机科学、商科、纳米技术等领域享有盛誉。学校拥有丰富的学术资源，包括W.E.B. DuBois图书馆等，并为学生提供直接参与研究的机会。UMass Amherst是麻省五校联盟的成员，为学生提供了丰富的学术和文化体验。

## 关键声明
- UMass Amherst成立于1863年，是麻省大学系统的旗舰校区。
- 学校拥有21,000名本科生和6,200名研究生，包括来自近100个国家的国际学生。
- UMass Amherst属于国家一级研究型大学，是麻省五校联盟和世界大学联盟的成员。
- 学校在2020年世界大学学术排名中位列全美第61名，2020年U.S. News美国最佳大学排名中位列第64位。
- UMass Amherst在计算机科学领域享有极高的声誉，在CSRankings中位列全美第12位，全球第16位。
- 学校提供广泛的本科和研究生课程，包括计算机科学、商科、纳米技术、聚合物科学、语言学和工程设计等。
- UMass Amherst拥有丰富的学术资源，包括W.E.B. DuBois图书馆等。
- 学校鼓励学生参与研究，并提供70多个国外学习及学生交流机会。
- UMass Amherst是麻省五校联盟的成员，与其他四所文理学院共享教学资源。
