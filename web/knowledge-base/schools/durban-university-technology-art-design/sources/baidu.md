# 德班理工大学 艺术设计

## TL;DR
德班理工大学是一所位于南非的技术性公立大学，成立于2002年，由纳塔尔理工大学和索尔塔理工大学合并而成。学校以技术培训为主导，设有会计与信息学、应用科学、艺术与设计等六大学院。艺术设计学院提供多个专业，包括建筑科技及室内设计、时装及纺织、美术和珠宝设计等。学校获得中国教育部认证，并与多所中国高校有合作项目。学费和生活费总计约24000-30000兰特/年。

## 关键声明
- **学校概况**：德班理工大学是南非一所技术性公立大学，成立于2002年，由纳塔尔理工大学和索尔塔理工大学合并而成。学校位于德班和彼得马里茨堡，在校生约23000人。
- **学院设置**：学校设有会计与信息学、应用科学、艺术与设计等六大学院。
- **艺术设计专业**：艺术设计学院提供建筑科技及室内设计、时装及纺织、美术和珠宝设计等专业。
- **认证与排名**：学校获得中国教育部认证，工程类课程通过南非工程理事会认证。
- **国际合作**：学校与中国高校开展教师互换、研究生交流及共建孔子学院等合作。
- **学费与生活费**：学费约8900-15000兰特/年，生活费约15000兰特/年，总计约24000-30000兰特/年。

## 艺术设计专业

- **专业介绍**：培养具备建筑科技和室内设计专业知识和技能的高级应用型人才。
- **课程设置**：建筑设计、室内设计、建筑制图、建筑力学、建筑材料等。
- **就业方向**：建筑设计公司、室内设计公司、装饰公司等。

### 时装及纺织
- **专业介绍**：培养具备时装设计、纺织工艺、服装制作等专业知识和技能的高级应用型人才。
- **课程设置**：时装设计、服装结构、服装材料、服装工艺、纺织工艺等。
- **就业方向**：服装设计公司、纺织企业、服装制作公司等。

### 美术和珠宝设计
- **专业介绍**：培养具备美术和珠宝设计专业知识和技能的高级应用型人才。
- **课程设置**：美术基础、珠宝设计、珠宝工艺、珠宝材料等。
- **就业方向**：珠宝设计公司、美术工作室、珠宝制作公司等。
