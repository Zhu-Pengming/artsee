# 威斯康星大学密尔沃基分校（艺术与设计学院）维基百科条目

## TL;DR
威斯康星大学密尔沃基分校（艺术与设计学院，简称MIAD）是一所位于美国威斯康星州密尔沃基的私立艺术学院，成立于1974年。学院规模较小，拥有约600名学生，占地面积2英亩。学院以提供艺术与设计相关教育而闻名，但在2015年并未被《美国新闻与世界报道》排名。

## 关键声明
- **成立时间**：1974年
- **学校类型**：私立大学
- **捐赠基金**：$270万（2011年）
- **校长**：Neil Hoffman
- **教师人数**：200人
- **本科生人数**：668人
- **校区面积**：2英亩（0.81公顷）
- **位置**：美国威斯康星州密尔沃基
- **排名情况**：2015年未在《美国新闻与世界报道》排名中
