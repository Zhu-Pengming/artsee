# 亚历山大大学美术学院概况

## TL;DR
亚历山大大学美术学院是埃及历史悠久的非宗教大学之一，始建于1938年，位于埃及亚历山大市。学院在阿拉伯国家享有盛誉，尤其在医学研究、艺术与科学领域。本文将详细介绍亚历山大大学美术学院的概况、历史沿革、学术排名、知名校友以及学院特色。

## 备注/分析
### 建校历史
亚历山大大学美术学院始建于1938年，是埃及历史悠久的非宗教大学之一。学院的前身是法鲁克大学（现为开罗大学）的一部分，1942年独立，1952年因埃及七月革命爆发，法鲁克王朝被推翻而改名。这一历史背景为学院的发展奠定了坚实的基础。

### 学术排名
亚历山大大学美术学院在阿拉伯国家享有盛誉，尤其在医学研究、艺术与科学领域。学院在多个国际排名中表现优异，如2019年USNEWS世界大学排名745名，2020年QS世界大学排名位列801-1000区间，在阿拉伯地区大学中排名16。

### 知名校友
亚历山大大学美术学院培养了众多知名校友，如诺贝尔化学奖得主艾哈迈德·泽维尔、阿拉伯埃及现代著名作家塔哈·侯赛因等。这些校友为学院赢得了极高的声誉。

### 学院特色
亚历山大大学美术学院拥有先进的教学设施和丰富的学术资源，为学生提供了良好的学习环境。学院注重培养学生的创新能力和实践能力，为学生提供了丰富的实践机会。

### 学校简介
亚历山大大学为埃及国立高等院校，建于1938年。亚历山大大学以医学研究，特别是解剖和外科手术而著称。学校现有教职工10089人，其中从事科研教授有2966人。截至2020年2月，亚历山大大学在校学生共有197278人，其中留学生3659人，硕士研究生9177人，博士研究生3178人，包括埃及本地、阿拉伯国家和其他国家的学生。

### 历史沿革
亚历山大大学美术学院的历史可以追溯到1938年，当时是法鲁克大学（现为开罗大学）的一部分。1942年独立，1952年因埃及七月革命爆发，法鲁克王朝被推翻而改名。学院在发展过程中，不断壮大，现已成为埃及乃至阿拉伯国家的重要高等学府。

- **2020年泰晤士高等教育世界大学影响力排名**：101—200

### 知名校友
- **艾哈迈德·泽维尔**：诺贝尔化学奖得主
- **塔哈·侯赛因**：阿拉伯埃及现代著名作家、文学家、文艺批评家
- **塞奥佐罗斯二世教皇**：亚历山大科普特正教会的第118任牧首

亚历山大大学美术学院位于埃及亚历山大市，拥有优美的校园环境和丰富的文化氛围。学院周边交通便利，生活设施完善，为学生提供了良好的学习和生活环境。
