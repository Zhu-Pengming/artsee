# 加州大学洛杉矶分校（艺术与建筑学院）- 播报

## TL;DR
本文详细介绍了加州大学洛杉矶分校（UCLA）的历史沿革、办学条件、学术研究、校园文化、行政管理、校园环境、校友情况、所获荣誉和学校事件。UCLA是一所位于美国洛杉矶的公立综合性研究型大学，被誉为“公立常春藤”。学校拥有丰富的学术资源和优秀的师资力量，提供多个本科和研究生专业，并在全球范围内享有盛誉。

## 关键声明

### 历史沿革
- UCLA成立于1919年，最初名为加州大学南部分校，后更名为加州大学在洛杉矶分校，最终定名为加利福尼亚大学洛杉矶分校。
- 学校发展历程中，曾成立多个学院和研究所，如建筑与城市规划学院、公共政策与社会研究学院、加州纳米系统研究所等。

### 办学条件
- UCLA设有13个学院，开设135个本科专业，提供130多个学术和专业领域的硕士和博士学位课程。
- 学校师资力量雄厚，拥有7位诺贝尔奖获得者、1位菲尔兹数学奖获得者、15位麦克阿瑟基金会研究员等。

### 学术研究
- UCLA拥有290多个研究机构，每年收到的研究经费超过10亿美元。
- 学校在多个领域取得重要科研成果，如胎儿监护技术、神经移植技术、艾滋病病例报告等。

### 校园文化
- UCLA拥有丰富的校园文化，包括学校标志、校训、校歌、吉祥物、校色等。
- 学校设有1200多个俱乐部和学生组织，为学生提供多样化的课外活动。

### 校友情况
- UCLA拥有超过660000名在世校友，校友会会员89164人，其中8位校友获得诺贝尔奖。

### 学校事件
- UCLA曾因支持巴勒斯坦抗议活动被指控容忍反犹太主义，后与特朗普政府达成和解协议。

- 建筑研究、艺术、舞蹈、设计媒体艺术、世界艺术与文化、个人专注领域（艺术与建筑）
- 专科学院（The College）
- 非裔美国人研究、古代近东和埃及学、人类学、应用语言学、天体物理学、气候科学、欧洲语言和跨文化研究、金融精算数学、语言学和西班牙语、海洋生物学等

### 学科建设
- UCLA在130多个学术和专业领域提供的硕士和博士学位课程。
- 其中，超过40个研究生课程在美国排名前10位，包括艺术、心理学、数学、历史、社会学和英语等。

### 师资力量
- UCLA共有教职人员5464人。
- 其中，有7位诺贝尔奖获得者，1位菲尔兹数学奖获得者，15位麦克阿瑟基金会研究员，12位国家科学奖章获得者，1位普利兹克奖获得者，3位普利策奖获得者，1位图灵奖获得者。
