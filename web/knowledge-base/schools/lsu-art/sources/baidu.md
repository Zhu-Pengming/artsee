# 路易斯安那州立大学介绍

## TL;DR
路易斯安那州立大学（Louisiana State University，简称LSU）是一所位于美国路易斯安那州巴吞鲁日的公立研究型大学，拥有超过150年的历史。学校占地面积广阔，被誉为全美最美校园之一。LSU提供230多个学科，涵盖艺术、工程、商业、法律等多个领域。学校在景观设计、石油工程等领域享有盛誉，并拥有强大的体育传统。本文将详细介绍LSU的历史、学院设置、师资力量、校园环境、学校排名、学科介绍、体育运动、知名校友以及申请相关信息。
