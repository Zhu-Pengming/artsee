# 路易斯安那州立大学

## TL;DR
路易斯安那州立大学（Louisiana State University, LSU）是一所位于美国路易斯安那州巴吞鲁日的公立旗舰州立大学，成立于1853年。该校拥有超过31,000名学生，提供广泛的本科和研究生课程。LSU在全球学术排名中表现良好，尤其在体育竞技方面享有盛誉，2019年美式橄榄球队赢得全国冠军。该校培养了许多知名校友，包括沙奎尔·奥尼尔和皮特·马拉维奇等体育明星。

## 备注/分析
### 学校概况
- LSU是一所历史悠久的公立大学，拥有超过150年的历史。
- 学校位于美国南部路易斯安那州，是该州最大的大学之一。
- LSU提供广泛的本科和研究生课程，满足不同学生的需求。

### 学生人数
- LSU拥有超过31,000名学生，包括26,159名本科生和5,368名研究生。
- 学生人数众多，学校提供丰富的学术资源和社交活动。

### 学术排名
- LSU在全球学术排名中表现良好，尤其在体育竞技方面。
- 学校的体育成就反映了其校园文化和学生活力。

### 体育成就
- 2019年，LSU美式橄榄球队赢得全国冠军，这是学校历史上的一大亮点。
- 体育成就是LSU校园文化的重要组成部分。

### 校友
- LSU培养了许多知名校友，包括沙奎尔·奥尼尔和皮特·马拉维奇等体育明星。
- 这些校友的成功故事激励着在校学生追求卓越。
