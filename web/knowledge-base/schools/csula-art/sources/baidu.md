# 加州州立大学洛杉矶分校介绍

## TL;DR
加州州立大学洛杉矶分校（CSULA）是一所位于美国加利福尼亚州洛杉矶的公立大学，成立于1947年。该校提供多种本科和研究生学位课程，涵盖艺术、商业、教育、工程、健康和公共事务等多个领域。CSULA以其艺术和公共事务学院而闻名，在US News排名中分别位列第60位和第97位。学校注重实用教育，培养了众多工程师和教师，是南加州地区高等教育的核心力量。

## 关键声明
- **成立时间与位置**：1947年，位于美国加利福尼亚州洛杉矶东部。
- **学校规模**：校园占地200多亩，拥有6个学院，50多个科系，学生人数超过20,000名。
- **教育特色**：注重实用教育，培养工程师和教师。
- **学位课程**：提供100种学士学位，70多种硕士学位，2种博士学位课程。
- **学院与专业**：包括商学院、教育学院、艺术学院、公共事务学院等，涵盖多个领域。
- **学校排名**：艺术学院排名第60位，公共事务学院排名第97位。
- **知名校友**：包括潘玮柏、Billie Jean King等。
- **入学要求**：学士学位要求高中毕业，GPA 2.7以上，托福550分；硕士学位要求学士学位，GPA 2.7以上，托福550分，GRE或GMAT成绩。
- **学费与费用**：本科学费/年$12,718，硕士学费/年$12,160，总费用/年$24,455（本科）和$24,965（硕士）。
- **录取率**：44.85%。

## 学校概况
- **成立时间与位置**：CSULA成立于1947年，位于美国加利福尼亚州洛杉矶东部。
- **学校规模**：校园占地200多亩，拥有6个学院，50多个科系，学生人数超过20,000名。
- **教育特色**：CSULA注重实用教育，培养工程师和教师，是南加州地区高等教育的核心力量。

## 学位课程
- **学士学位**：CSULA提供100种学士学位课程，涵盖艺术、商业、教育、工程、健康和公共事务等多个领域。
- **硕士学位**：学校提供70多种硕士学位课程，包括教育、工程、商业、艺术等领域。
- **博士学位**：CSULA提供2种博士学位课程，包括教育管理与领导学博士和特殊教育学博士。

## 学院与专业
- **商学院**：提供会计、经济与统计、金融和法律、信息系统、管理学、市场营销等专业。
- **教育学院**：提供教育应用与发展、课程与指导、特殊教育与咨询服务等专业。
- **艺术学院**：提供艺术、传媒、文学研究、音乐、哲学、表演艺术等专业。
- **公共事务学院**：提供公共管理与社会服务学、社会学、政治科学与政府学等专业。
- **工程与计算机科学技术学院**：提供土木工程、计算机科学、电气与计算机工程、机械工程、工程技术等专业。
- **健康与公共事业服务学院**：提供儿童与家庭研究、犯罪学与刑事学、保健科学、运动机能学和营养学、护理学、社会服务等专业。
- **自然与社会科学学院**：提供人类学、生物学、化学与生物化学、地理与城市研究、地质学、历史、数学、物理和天文学、心理学、社会学等专业。
