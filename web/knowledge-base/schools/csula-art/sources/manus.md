## 关键声明

### 学校概述

- Cal State LA 是一所公立综合大学，属于加州州立大学系统。
- 学校成立于1947年，位于洛杉矶东部University Hills，紧邻Alhambra等华人聚居区。
- 学校总学生数约22,000-26,000人，师生比为22:1。

### 项目数据

- Cal State LA 艺术系提供本科BA、硕士MA和MFA项目。
- BA项目包括动画、艺术教育、艺术史、设计、时尚纤维材料、工作室艺术等方向。
- MA项目侧重学术研究，MFA项目侧重创作实践。
- 作品集要求根据项目不同而有所差异。

### 申请要求

- 申请本科需要满足英语语言要求，包括TOEFL、IELTS、Duolingo等。
- 研究生申请需要满足更高的英语语言要求，并提供作品集。
- 申请费为70美元。

### 学费与生活成本

- 本科学费为7,164美元/年（加州居民）或19,764美元/年（非加州/国际生）。
- 研究生学费为8,688美元/年（加州居民）或18,768美元/年（非加州/国际生）。
- 生活成本因人而异，但预计每年约34,522美元（校内住宿）或47,122美元（校外住宿）。

### 校园设施与创作资源

- Cal State LA 艺术系拥有Fine Arts Building、Luckman Fine Arts Complex等设施。
- 系内拥有Ronald H. Silverman Fine Arts Gallery、COMA Gallery等展览空间。
- 学生可以使用陶艺、绘画、摄影、版画、雕塑、纤维/时尚等工作室。
