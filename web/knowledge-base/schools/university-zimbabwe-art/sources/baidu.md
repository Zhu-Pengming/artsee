# 津巴布韦大学 艺术系

## TL;DR
津巴布韦大学是津巴布韦第一所大学，成立于1955年，位于首都哈拉雷。该校是津巴布韦最著名的综合性大学，拥有11个学院和多个学士学位及硕士学位课程。本文源提供了该校的基本信息，包括历史、地理位置、学院设置和学术排名。

## 关键声明
- **成立时间与历史**：津巴布韦大学成立于1955年，最初名为罗德西亚和尼亚萨兰大学学院，隶属于伦敦大学。1980年津巴布韦独立后，更名为津巴布韦大学。
- **地理位置**：大学位于津巴布韦首都哈拉雷，占地面积299公顷。
- **学院设置**：大学设有11个学院，包括理学院、社会学院、文学院、商学院、法学院等。
- **学术排名**：2025年泰晤士高等教育世界大学排名位列1501+。
- **孔子学院**：津巴布韦大学孔子学院是非洲建立最早的孔子学院之一，成立于2007年。
