# 墨西哥瓜纳华托大学简介

## TL;DR
墨西哥瓜纳华托大学是一所位于墨西哥瓜纳华托州的公立大学，提供从高中到博士阶段的教育。在校学生约31,000人，其中国际学生约250人。大学在2026年QS世界大学排名中位列1401+名，拥有多个校区，包括主校区塞拉亚-萨尔瓦铁拉和莱昂校区。大学在文化与艺术领域提供学士学位项目，并与中国高校有学术合作。此外，大学在2022年创下了吉尼斯世界纪录，制作了世界上最大杯的咖啡。

## 关键声明
### 学校概况
- **类型**: 公立大学
- **学生人数**: 约30,992人（85%为本科生）
- **国际学生人数**: 251人
- **教职员工人数**: 1,538人（5%为国际教职员工）
- **QS排名**: 2026年位列1401+名
- **校区**: 塞拉亚-萨尔瓦铁拉（主校区），莱昂校区

### 学术合作
- 与中国武汉科技大学、武汉工程大学在水资源评价、矿物加工等领域开展学术合作及联合培养

### 世界纪录
- 2022年12月10日，学生制作了容量26,939.22升的咖啡，创吉尼斯世界纪录
