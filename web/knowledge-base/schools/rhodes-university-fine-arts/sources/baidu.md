# 罗格斯大学介绍

## TL;DR
罗格斯大学（Rutgers University）是美国新泽西州规模最大的高等学府，拥有悠久的历史和卓越的学术研究实力。本文将详细介绍罗格斯大学的历史沿革、办学条件、学术研究、校园文化、校区情况、行政管理、校园环境、校友情况以及所获荣誉等方面。

## 关键信息

### 历史沿革
- 罗格斯大学成立于1766年，原名为皇后学院，是美国独立战争前建立的九所殖民地学院之一。
- 1825年，学院更名为罗格斯学院，以纪念慈善家亨利·罗格斯上校。
- 1864年，罗格斯学院成为新泽西州立赠地学校，正式成立罗格斯科学学院。
- 1924年，罗格斯科学学院更名为罗格斯大学。
- 罗格斯大学是美国大学协会成员之一，也是“公立常春藤”大学之一。

### 办学条件
- 罗格斯大学设有29个学院，包括梅森格罗斯艺术学院、罗格斯商学院、文理学院、传播与信息学院、工程学院等。
- 学校拥有10000余名全职和兼职教师，截至2024年6月，共有67620名学生。
- 罗格斯大学开设了150多个本科专业，400多个硕士和博士学位。

### 学术研究
- 罗格斯大学设有300多个科研中心和研究所，涵盖多个学科领域。
- 学校在数学、物理、统计学、信息科学、计算机科学等领域拥有一批世界著名的学者与研究员。
- 罗格斯大学近年获得多项学术奖项，包括诺贝尔奖、普利策奖、菲尔兹奖等。

### 校园文化
- 罗格斯大学的校训为“正义之光，耀我西方”。
- 学校拥有丰富的校歌传统，包括《在古老的拉里坦河畔》、《钟声必须敲响》等。
- 罗格斯大学的校徽体现了学校的历史和价值观。

### 校区情况
- 罗格斯大学由新不伦瑞克校区、纽瓦克校区、卡姆登校区和罗格斯大学生物医学与健康科学中心组成。
- 新不伦瑞克校区是主校区，拥有多个分校区，包括College Avenue、Busch Campus、Livingston Campus、Douglass Campus、Cook Campus等。
- 纽瓦克校区以商学院和医学院为主，MBA总部及公共事务与管理学院亦位于此校区。
- 卡姆登校区开设有文理学院、成人高校、研究生院、商学院、法学院、护理学院以及艺术学院。

### 行政管理
- 罗格斯大学由理事会管理，理事会由15名有投票权的成员组成。
- 乔纳森·霍洛威（Jonathan Holloway）是罗格斯大学的校长。

### 校园环境
- 罗格斯大学的校园面积极大，共分为三个校区，主要分布于新泽西州北、中、西部三个区域。
- 校园环境优美，充满活力，拥有多个图书馆、博物馆、艺术中心等设施。

### 校友情况
- 罗格斯大学培养了众多杰出校友，包括诺贝尔奖得主、普利策奖得主、商界领袖、政治家等。

### 所获荣誉
- 罗格斯大学被《福布斯》评为新泽西州最佳雇主之一。
