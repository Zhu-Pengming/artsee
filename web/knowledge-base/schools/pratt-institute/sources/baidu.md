# 普瑞特艺术学院介绍

## TL;DR
普瑞特艺术学院是一所位于美国纽约市的著名私立美术学院，成立于1887年。学院在艺术与设计领域享有盛誉，提供多种本科和研究生课程，并与多家知名企业合作。本文将详细介绍普瑞特艺术学院的学院简介、数据、下设学院、提供专业、学院声誉、学术交流以及主楼火灾等内容。

## 关键声明
- **成立时间与地点**：普瑞特艺术学院成立于1887年，位于美国纽约市。
- **排名与声誉**：2023年QS世界大学排名中，艺术与设计类专业排名全球第六，全美第三。
- **学院规模**：拥有近50个本科和硕士学位课程，下设六个学院。
- **合作与资源**：与苹果、蔻驰等企业建立课程合作，并与视觉艺术学院、帕森斯设计学院共享师资资源。
- **校区分布**：在纽约布鲁克林、曼哈顿及尤蒂卡设有校区，布鲁克林校区配备尖端校园设施。
- **教育理念**：致力于培养学生将艺术热情转化为创造性实践，注重多样性和包容性。
- **学术交流**：与世界多所美术学院有合作关系，并提供交换生项目。
- **火灾事件**：2013年2月15日，学院主楼发生火灾，导致42间艺术工作室及大量学生作品损毁。

## 学院简介
普瑞特艺术学院是一所位于美国纽约市的著名私立美术学院，成立于1887年。学院在艺术与设计领域享有盛誉，提供多种本科和研究生课程，涵盖了艺术、设计、建筑等多个领域。学院致力于培养学生将艺术热情转化为创造性实践，注重多样性和包容性。

## 学院数据
- **成立时间**：1887年
- **地点**：美国纽约市
- **排名**：2023年QS世界大学排名中，艺术与设计类专业排名全球第六，全美第三
- **课程数量**：近50个本科和硕士学位课程
- **校区**：纽约布鲁克林、曼哈顿及尤蒂卡
- **师生比**：10:1
- **国际学生比例**：32%

## 下设学院
- **建筑学院**：提供建筑学、城市设计、建筑与设施管理、环境管理、城市场所建设、历史保护与房地产等专业领域的本科和硕士学位。
- **艺术学院**：提供艺术与设计教育、数字艺术、2D动画、3D动画、交互艺术、游戏艺术、电影、纯艺术、绘画、珠宝、版画、雕塑与综合实践、摄影等专业领域的本科和硕士学位。
- **设计学院**：提供传达设计、平面设计、插画、服装设计、工业设计、室内设计、室内设计与包装、身份与系统设计等专业领域的本科和硕士学位。
- **继续教育学院**：提供各种课程和项目，满足不同层次学生的需求。
- **文理学院**：提供人文和媒体研究、数学和科学、社会科学和文化研究以及艺术和设计史的课程。
- **信息学院**：提供图书馆与信息科学、博物馆与数字文化、信息体验设计以及数据分析与可视化等方面的研究生课程。

## 主楼火灾
2013年2月15日，普瑞特艺术学院主楼发生火灾，导致42间艺术工作室及大量学生作品损毁。火灾起因可能是主楼木质屋顶上的电线失火。
