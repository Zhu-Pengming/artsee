# 亚利桑那州立大学概况

## TL;DR
亚利桑那州立大学（ASU）是一所位于美国亚利桑那州的公立研究型大学，拥有悠久的历史和卓越的学术声誉。学校提供多种本科和研究生课程，拥有来自世界各地的学生和教师。ASU以其创新的教育模式、强大的科研能力和广泛的国际合作而闻名。

## 关键声明
- ASU成立于1885年，是一所公立研究型大学。
- 学校在2024QS世界大学排名中位列全球第179位。
- ASU拥有来自157个国家和地区的12843名国际学生。
- 学校提供400多个本科专业和150个研究生专业。
- ASU拥有5个主校区和4个分校区，并在加利福尼亚州和华盛顿特区设有学习中心及研究所。
- 学校拥有5300名教职工，包括5名诺贝尔奖、11名普利策奖得主以及66名国家院士。
- ASU在可持续发展、环境保护和科技创新方面表现出色。

## 学校概况
### 历史沿革
亚利桑那州立大学成立于1885年，最初是一所师范学校。经过多年的发展，学校已成为一所综合性的研究型大学，拥有多个学院和研究所。

### 办学条件
- **院系设置**：ASU拥有18个学院和12所学校，提供400多个本科专业和150个研究生专业。
- **学科建设**：ASU在多个学科领域拥有卓越的学术声誉，尤其是在可持续发展和科技创新方面。
- **师资力量**：ASU拥有5300名教职工，包括5名诺贝尔奖、11名普利策奖得主以及66名国家院士。
- **教学建设**：ASU提供多种本科和研究生课程，包括在线课程和在职教育项目。
- **合作交流**：ASU与全球多个高校和研究机构建立了合作关系，为学生提供了丰富的学习和交流机会。

### 学术研究
- **科研平台**：ASU拥有多个科研平台，包括Skysong Innovations、亚利桑那州立大学研究园等。
- **研究成果**：ASU在多个领域取得了重要的科研成果，包括可持续发展、环境保护和科技创新。
- **学术资源**：ASU拥有丰富的学术资源，包括图书馆、期刊、数据库等。

### 校区情况
- **坦佩校区**：ASU的主校区位于坦佩市，拥有多个学院和研究所。
- **理工校区**：ASU的理工学院校园位于凤凰城西部边缘，是跨学科科学、工程、管理、技术和教育研究的中心。
- **西校区**：ASU的西校区位于凤凰城西部边缘，是新跨学科艺术与科学学院的所在地。
- **哈瓦苏湖城校区**：ASU的哈瓦苏湖城校区位于哈瓦苏湖，提供多种本科和研究生学位课程。
- **凤凰城市中心校区**：ASU的凤凰城市中心校区位于亚利桑那州首府，为学生提供多种学术和专业联系。
- **梅奥医药研究所**：ASU与马约诊所在亚利桑那州的斯科茨代尔合作开办医药研究所。
- **亚利桑那州立大学研究园**：ASU研究园为51家企业租户提供支持，雇用超过6,000名员工。
- **Novus创新走廊**：Novus创新走廊是全国最先进的城市综合开发项目之一，毗邻ASU坦佩校区。
- **亚利桑那州立大学美术馆陶艺中心**：ASU美术馆陶艺中心举办了多场国际陶瓷艺术展览。

### 校友情况
ASU拥有众多杰出校友，包括政治家、企业家、学者等。

### 所获荣誉
ASU在多个领域获得了荣誉和奖项，包括美国创新排名第一、美国排名第一、全球排名前十等。
