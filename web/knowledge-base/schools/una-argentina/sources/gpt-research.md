# 阿根廷国立艺术大学申请研究报告

## TL;DR
Universidad Nacional de las Artes（UNA）是阿根廷一所公立艺术大学，提供从预科到博士的学位课程。学校采用分布式校园模式，分布在布宜诺斯艾利斯市内多个地点。UNA 对国际学生开放，但申请流程和费用结构较为复杂。以下是对UNA申请流程、学位体系、费用、生活成本、国际合作等方面的详细分析。

## 关键主张

### 学校概况
- UNA是一所公立艺术大学，前身可追溯至1996年的Instituto Universitario Nacional del Arte（IUNA）。
- 学校采用分布式校园模式，分布在布宜诺斯艾利斯市内多个地点。
- 学校提供从预科到博士的学位课程，涵盖艺术、设计、音乐、戏剧等多个领域。

### 申请流程
- 国际学生需要处理外国中学学历的认可或同等手续。
- 本科申请以完成中学教育并完成各院系入门课程/分流课程为主。
- 研究生申请通常要求至少四年制大学或高等非大学学历。
- 学费结构复杂，本科通常免费，研究生项目则根据项目和专业有所不同。

### 学位体系
- UNA提供预科/本科、专门化、硕士和博士等多个学位层级。
- 本科和预科课程涵盖影视、舞蹈、戏剧、多媒体、音乐、视觉艺术、艺术批评等多个领域。
- 研究生课程包括艺术、文化研究、艺术教育、舞蹈、音乐研究、批评与艺术史、多媒体声音等多个领域。

### 费用和生活成本
- 本科通常免费，但可能涉及行政性杂费。
- 研究生项目明确收费，且项目差异大。
- 生活成本方面，学校与政府未提供UNA专属统一月预算。

### 国际合作
- UNA与多个国家和地区的高校有合作关系。
- 学校提供交换项目，允许学生到其他学校学习。

### 校园设施
- 学校提供多个设施，包括博物馆、图书馆、工作室等。

### 常见问题
- 能否全英文申请？学校未给出统一英文申请通道，本质上仍以西语运作为主。
- 国际本科新生能否立刻申请国家奖学金Progresar？一般不能把它当首年确定性资金，因为官方要求高教类别外国申请人须有至少5年合法居留。
- 是否所有项目都要求作品集？不是；研究型硕士明确偏向研究计划/动机信，但表演、舞蹈、音乐、视觉等项目的专业性测试可能散落在院系导入环节里，必须逐项目查。
