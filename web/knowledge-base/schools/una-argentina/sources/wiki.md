# 阿根廷国立艺术大学 - 维基百科

## TL;DR
阿根廷国立艺术大学（Universidad Nacional de las Artes）是一所位于布宜诺斯艾利斯的公立艺术和研究型大学，拥有悠久的历史和丰富的学术资源。该校提供涵盖艺术各领域的学术教育，包括学士、硕士和博士学位。本文将详细介绍该校的历史、部门设置、学术人员、学生人数、地理位置、以及杰出校友等信息。

## 关键声明
- **成立时间**：阿根廷国立艺术大学的历史可以追溯到1875年，经过多次更名和发展，最终在2014年更名为现名。
- **类型**：公立艺术和研究型大学。
- **地理位置**：位于阿根廷首都布宜诺斯艾利斯。
- **学术人员**：拥有1,703名学术人员。
- **学生人数**：拥有16,806名学生。
- **部门设置**：包括视听艺术、戏剧艺术、动力艺术、音乐、视觉艺术、艺术批评、民俗学、多媒体、写作艺术和教育工作者计划等。
- **学位授予**：提供学士、硕士和博士学位。
- **杰出校友**：包括音乐评论家、电台主持人、音乐作家爱德华多·阿诺西，以及戏剧、电影和电视演员安吉拉·拉格诺等。
