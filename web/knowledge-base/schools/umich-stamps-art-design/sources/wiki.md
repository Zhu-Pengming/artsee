# 密歇根大学安娜堡分校维基百科条目

## TL;DR
密歇根大学安娜堡分校是一所成立于1817年的美国顶尖公立研究型大学，位于密歇根州安阿伯市。该校是美国大学协会创始成员之一，在全球多个大学排名中位居前30位，被誉为"西部的哈佛大学"和"美国公立大学的母亲"。密歇根大学拥有19个学术学院，开设15,000多门课程，200多个专业，其中多个学科在全美排名前10。学校拥有庞大的学生群体（约53,488人），国际学生占比约7.4%，培养了26名诺贝尔奖得主和众多知名校友。校园分为北校区、中校区和南校区三个主要部分，总面积达84.84平方公里。学校以其强大的体育传统著称，狼獾队在NCAA各项赛事中取得了卓越成就。

## Key Claims

### 学校概况与历史背景
- **基本信息**：密歇根大学成立于1817年8月26日，校训为"Artes, Scientia, Veritas"（艺术、知识、真理），位于美国密歇根州安阿伯市，是一所公立研究型大学，为美洲大学协会的创始院校之一。
- **学校规模**：截至2025年，学校共有学生53,488人，其中本科生35,358人，研究生18,130人；教师8,526人；职工24,765人。捐赠基金达212亿美元，预算156亿美元。
- **地理位置**：安阿伯是一个典型的大学城，在全美最适宜居住的城市中排名第七。校园分为北校、中校、南校三个核心部分，总面积85平方公里，核心面积13平方公里。
- **历史沿革**：学校最初称为"普世知识学院"（Catholepistemiad），后改称"密歇根尼亚大学"，1838年开始使用"密歇根大学"名称至今。学校与美国共济会有密切关联，多位创办人是共济会会员。
- **发展里程碑**：
  - 1841年正式开始授课
  - 1855年成为美国第二所颁授理学学士学位的大学
  - 1900年与其他12所大学共同创立美国大学协会
  - 二战后承担多项联邦政府资助的研究工作，帮助发展和平用途的核能技术

### 学术实力与排名
- **综合排名**：根据《US News》、世界大学学术排名、QS世界大学排名、泰晤士高等教育世界大学排名等主流学术排名，密歇根大学的综合排名位居世界前30位、全美前30位。
- **学科优势**：
  - 医学院、法学院、工学院和教育学院都排在《US News》全美前10名
  - 大部分专业都在全美前20名
  - 在泰晤士报评出的声誉排名中位列世界第15位
  - 音乐戏剧学院位列美国音乐学院第四名
- **研究实力**：
  - 2020年自然指数报告，研究输出位居美国第5位
  - 大学研究经费数额全美第2
  - 美国联邦政府对大学研究资助数额全美第3
  - 在政治学和心理学上，"密歇根学派"与芝加哥学派、哥伦比亚学派齐名
- **特色研究项目**：
  - 密歇根大学消费者信心指数（罗斯商学院成果）
  - 中国研究中心（美国地理学家协会合作伙伴）
  - 肾上腺癌研究（世界顶尖水平）

### 学院设置与专业
- **学院架构**：密歇根大学共有19所学术学院，文理艺术学院为最大的学院，下设自然科学和社会科学多个部门。
- **主要学院**：
  - 文理学院（1841年成立）
  - 医学院（1850年成立）
  - 工程学院（1854年成立）
  - 法学院（1859年成立）
  - 牙科学院（1875年成立）
  - 药学院（1876年成立）
  - 音乐戏剧学院（1880年成立）
  - 护理学院（1893年成立）
  - 建筑和城市规划学院（1906年成立）
  - 商学院（1924年成立）
  - 艺术与设计学院（1974年成立）
- **专业设置**：
  - 开设15,000多门学士课程，200多个专业
  - 工学院、医务管理、公共卫生、心理学、药学、政治学、图书信息学、社会学及社会工作均列《US News》美国国内大学的前5位
  - 在《US News》全美前10名的专业包括教育学、工学、医学、护理学、法学、公共事务、公共卫生、心理学、社会学、政治学、经济学、统计学、历史学、化学、地球科学、英文和药学等
- **语言教学**：开设英语和各种外语课程，包括65门语言的教学，要求大学生必须掌握一门外语才能毕业。

### 校园设施与资源
- **校园布局**：
  - 中校区：历史街区，19至20世纪复兴式建筑，维多利亚时代建筑，装饰风艺术
  - 北校区：现代主义建筑，拥有卢丽塔（Lurie Tower）
  - 南校区：体育项目场地，包括密歇根体育场、克里斯勒中心和约斯特冰上竞技场
- **图书馆系统**：
  - 藏书830万册，每年增加17万册，是美国第七大的大学图书馆
  - 拥有全美最齐全的数学及牙医学的资料收藏
  - 西半球最多的古代纸草写文的收集
  - 亚洲、南亚洲、东南亚、斯拉维亚及东欧等的文件收集之齐全也是全美排名前5名
- **主要图书馆**：
  - 哈兰·哈彻研究生图书馆
  - 夏皮罗图书馆
  - 塔布曼医学图书馆

  - 音乐图书馆
  - 法学院图书馆
  - 克雷斯基工商管理图书馆
- **博物馆系统**：拥有九个主要的博物馆，包括凯尔西建筑博物馆、欣德丘兹牙医博物馆、密歇根大学植物标本博物馆、密歇根大学艺术博物馆、密歇根大学自然历史博物馆等。
- **体育设施**：
  - 密歇根体育场（西半球最大的体育场，世界第四大体育场）
  - 克里斯勒中心（篮球场）

  - 菲尔丁·约斯特冰上竞技场
  - 高尔夫球场（由阿利斯特·麦肯齐设计）
