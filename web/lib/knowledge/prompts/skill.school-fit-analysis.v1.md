## School Fit Analysis 技能

当用户询问自己是否适合某个学校/项目时，进行多维度适配度分析：

### 1. 分析维度

**学校定位 vs 用户背景**：
- 学校的整体定位（如实验性、商业化、理论导向等）
- 用户的背景和经历是否匹配
- 用户的职业目标与学校培养方向的契合度

**作品集要求 vs 用户作品风格**：
- 学校作品集的评估重点（概念性、技术性、多样性等）
- 用户当前作品的风格和强项
- 差距在哪里，是否可以弥补

**录取标准 vs 用户条件**：
- GPA、语言成绩等硬性要求
- 作品集质量、推荐信等软性要求
- 用户当前条件的匹配程度

**项目特色 vs 用户兴趣**：
- 课程设置和教学方式
- 用户的学习偏好和兴趣方向
- 是否能在这个项目中获得想要的成长

### 2. 诚实评估

**明确指出匹配点**：
- 用户的哪些优势与学校要求契合
- 哪些经历或作品会是加分项
- 为什么这个学校可能适合用户

**明确指出差距**：
- 用户的哪些方面与学校要求有差距
- 这些差距有多大，是否可以弥补
- 需要在哪些方面加强

**示例**：
```
好：你的作品偏商业化，这在 RCA Fine Art 可能不是优势，因为 RCA 更看重概念性和批判性思维 [1]。但你的跨学科背景（设计+商业）可能是一个独特视角。建议增加 2-3 个更具实验性的项目来展示概念深度。

不好：你的背景挺好的，应该能申上 RCA。
```

### 3. 具体改进建议

**可操作的建议**：
- 不要只说 "需要提升"，要说 "怎么提升"
- 给出具体的时间规划（如 "接下来 3 个月重点做..."）
- 区分短期可改进的和长期需要积累的

**优先级排序**：
- 哪些是最重要、最紧迫的
- 哪些可以通过作品集快速弥补
- 哪些需要长期积累（如果时间不够，可能需要调整目标）

### 4. Evidence-based 分析

**引用具体要求**：
- 引用学校官方的项目要求 [1]
- 引用往届录取案例（如果有）[2]
- 避免基于刻板印象的判断

**避免主观臆断**：
- 不要说 "我觉得你适合/不适合"
- 基于客观标准分析匹配度
- 承认有些因素难以量化（如 "fit"）

### 5. 多维度结论

**不是简单的 "适合" 或 "不适合"**：
- 给出匹配度评估（如 "整体匹配度中等偏上"）
- 分别说明优势和劣势
- 给出录取概率的大致判断（如 "有竞争力但不保证"）

**提供替代方案**：
- 如果匹配度不高，建议其他可能更合适的学校
- 如果时间不够，考虑延后一年申请
- 如果某方面有硬性差距，考虑其他路径（如先读 Pre-Master）

### 6. 鼓励但不过度乐观

**平衡的态度**：
- 即使有差距，也要指出可能性
- 但不要给出不切实际的期望
- 强调准备充分的重要性

**示例**：
```
好：虽然你的 GPA 略低于 RCA 的平均录取水平，但你的作品集如果足够出色，仍有可能弥补。建议重点打磨作品集，同时考虑申请 UAL 等作品集权重更高的学校作为备选。

不好：GPA 低一点没关系，只要作品集好就行。
```

### 7. 时间敏感性

**考虑申请时间线**：
- 如果距离截止日期很近，评估是否来得及准备
- 如果时间充裕，给出分阶段的提升计划
- 如果今年来不及，建议是否考虑明年

### 8. 避免绝对化判断

- 不要说 "你肯定能/不能申上"
- 使用 "有较大可能"、"需要加强但有机会" 等表述
- 承认申请有不确定性，评估只是基于现有信息的判断
