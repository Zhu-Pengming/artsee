## Hard Data 查询技能

当用户询问具体的硬数据（学费、截止日期、排名、官网等）时，遵循以下规范：

### 1. 数字抽取规范

**学费**：
- 必须包含货币符号和完整范围（如 "£31,350-39,750/年"）
- 避免模糊表述（如 "大概3万多"、"三万左右"）
- 如果是多年前的数据，必须标注年份（如 "2023学年学费为..."）
- 如果有不同项目的不同学费，分别列出，不要混淆

**其他数字**：
- GPA/成绩要求：保留原始格式（如 "3.0/4.0" 或 "80/100"）
- 录取率：使用百分比（如 "15%"），避免分数形式
- 学生人数：使用完整数字（如 "约2,500人"）

### 2. 日期抽取规范

**申请截止日期**：
- 必须包含完整日期（如 "2024年1月15日"）
- 避免相对时间（如 "下个月"、"年底前"）
- 标注是第几轮（如 "第一轮DDL: 2024年1月15日"）
- 如果有多个截止日期，按时间顺序列出

**其他时间信息**：
- 开学日期：完整月份和日期
- 课程时长：明确单位（如 "2年制"、"18个月"）

### 3. 链接抽取规范

**官网链接**：
- 必须是完整可点击的URL
- 避免模糊表述（如 "请访问官网"、"可以在官网查看"）
- 优先使用知识库中已验证的链接
- 如果有多个相关链接，标注用途（如 "官网首页"、"申请页面"、"学费说明"）

### 4. 来源标注

- 所有硬数据必须标注来源 [1][2]
- 如果数据来自不同来源且有冲突，明确指出（如 "根据[1]为£32,000，但[2]显示为£31,500"）
- 如果数据可能已过期，提醒用户核实（如 "以上为2023年数据，建议访问官网确认最新信息"）

### 5. 不确定性处理

如果参考资料中没有相关硬数据：
- 明确告知 "我暂时没有[具体学校/项目]的[具体数据]"
- 建议用户访问官网或联系学校
- **绝对禁止**编造数字、日期或链接
- **绝对禁止**使用模糊估算（如 "大概"、"可能"、"一般"）

### 6. 格式示例

**好的回答**：
```
RCA Fine Art MA 2024-2025学年学费为 £32,100/年 [1]。
第一轮申请截止日期为 2024年1月10日，第二轮为 2024年3月13日 [2]。
官方申请页面：https://www.rca.ac.uk/study/programme-finder/fine-art-ma/ [1]
```

**不好的回答**：
```
RCA纯艺大概三万多英镑一年，截止日期一般在年初，具体可以去官网看看。
```
